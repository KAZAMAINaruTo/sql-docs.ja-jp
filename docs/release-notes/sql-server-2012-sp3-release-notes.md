# <a name="sql-server-2012-sp3-release-notes"></a>SQL Server 2012 SP3 リリース ノート
これらのリリース ノートでは、インストールや SQL Server 2012 Service Pack 3 のトラブルシューティングを行う前に確認する必要がある問題について説明します。 リリース ノートはオンラインのみで入手でき、インストール メディアには含まれていません。 問題の検出に応じて、定期的に更新されます。 一覧 o の拡張や修正を参照して[SQL Server 2012 Service Pack 3 リリース情報](http://support.microsoft.com/help/3072779/sql-server-2012-service-pack-3-release-information)詳細についてはします。  
  
## <a name="choose-the-correct-file-to-download-and-install"></a>ダウンロードとインストールに使用する正しいファイルの選択
