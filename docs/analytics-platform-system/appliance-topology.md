# <a name="appliance-topology---analytics-platform-system"></a>アプライアンス トポロジ - 分析プラットフォーム システム
分析プラットフォーム システム全体の概要情報を表示します。  
  
全体のアプライアンスに適用される情報を構成するには、このセクションの要素を展開します。 アプライアンスの領域だけに適用される情報を構成する並列データ ウェアハウス トポロジまたは HDInsight トポロジのセクションを選択します。
