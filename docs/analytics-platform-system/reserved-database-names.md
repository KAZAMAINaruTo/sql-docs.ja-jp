# <a name="reserved-database-names"></a>予約済みのデータベース名
予約済みのデータベース名は、SQL Server PDW のユーザー データベースに割り当てることができない名前です。  
  
予約されているデータベース名のリスト:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
