# <a name="device-driver-control-error"></a>デバイス ドライバー制御エラー
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  このルールでは、システム イベント ログの EventId 11 を確認します。 このエラーは、デバイス ドライバーの破損、ハードウェアの問題、ケーブル接続の不具合、または接続の問題が原因で発生する場合があります。  
  
## <a name="best-practices-recommendations"></a>ベスト プラクティスと推奨事項  
 このエラーのトラブルシューティング方法の詳細については、次の Microsoft サポート技術情報を参照してください。  
  
-   [サポート技術情報の資料 259237](http://go.microsoft.com/fwlink/?linkid=117746)  
  
-   [サポート技術情報の資料 154690](http://go.microsoft.com/fwlink/?linkid=117747)
