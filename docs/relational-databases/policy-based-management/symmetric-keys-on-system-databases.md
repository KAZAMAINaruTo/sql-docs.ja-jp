# システム データベースの対称キー
  このルールでは、master、msdb、model、および tempdb の各データベースにあるユーザーが作成した対称キーについて確認します。  
  
## ベスト プラクティスと推奨事項  
 システム データベースには対称キーを作成しないでください。
