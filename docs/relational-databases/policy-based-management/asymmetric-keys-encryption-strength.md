# <a name="asymmetric-keys-encryption-strength"></a>非対称キー暗号化の強度
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] このルールでは、非対称キーが 1024 ビット以上の強度の暗号化を使用して作成されたかどうかを確認します。  
  
## <a name="best-practices-recommendations"></a>ベスト プラクティスと推奨事項  
 データ暗号化用の非対称キーを作成する場合は、1024 ビットの RSA 暗号以上の強度の暗号化を使用します。
