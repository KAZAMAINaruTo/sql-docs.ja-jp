# <a name="input-and-output-error-during-hard-page-fault"></a>ハード ページ フォールト中の I/O エラー
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] このルールでは、システム イベント ログの EventId 51 を確認します。 このエラーは、ハード ページ フォールト中のエラーが原因で発生します。  
  
## <a name="best-practices-recommendations"></a>ベスト プラクティスと推奨事項  
 このエラーのトラブルシューティング方法の詳細については、次の Microsoft サポート技術情報を参照してください。  
  
-   [サポート技術情報の資料 305547](http://go.microsoft.com/fwlink/?linkid=117748)
