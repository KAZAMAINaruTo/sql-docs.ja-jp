## <a name="explanation"></a>説明  
 ローカル データベース インスタンスに指定した共有名は既に使用されています。  
  
## <a name="user-action"></a>ユーザーの操作  
 Local Database Runtime インスタンスに別の名前を指定するか、既存のインスタンスを削除してください。
