# SQL Server レプリケーションにおける重大な変更
  このトピックで互換性に影響する変更について説明 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] レプリケーション。 これらの変更によって、以前のバージョンの [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]に基づくアプリケーション、スクリプト、または機能が使用できなくなる場合があります。 この問題は、アップグレードするときに発生することがあります。  
  
## 行われた変更は互換性に影響します。 [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)]  
 [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] レプリケーションをサポートしない [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] または [!INCLUDE[ssEW](../../includes/ssew-md.md)]です。
