# <a name="replication-monitor-main-page"></a>レプリケーション モニター、メイン ページ
  レプリケーション モニターを使用すると、レプリケーション トポロジにおけるパブリケーションとサブスクリプションの状態およびパフォーマンスを追跡できます。 以降のトピックで詳しく説明します。  
  
-   レプリケーション モニターの概要については、「[レプリケーションの監視](../../relational-databases/replication/monitor/monitoring-replication-overview.md)」を参照してください。  
  
-   レプリケーション モニターの左ペインには、パブリッシャーおよびパブリッシャーのグループが表示されます。 レプリケーション モニターに 1 つ以上のパブリッシャーを追加すると、パブリケーションとサブスクリプションの情報が表示されます。 詳細については、「 [レプリケーション モニターのパブリッシャーの追加および削除](../../relational-databases/replication/monitor/add-and-remove-publishers-from-replication-monitor.md)」を参照してください。
