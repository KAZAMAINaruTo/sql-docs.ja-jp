# [パブリッシャーのプロパティ - パブリッシャー]、[サブスクライバー]
   **サブスクライバー** のページ、 **パブリッシャーのプロパティ** ] ダイアログ ボックスの使用のバージョンを実行しているパブリッシャーの [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] より前のバージョン [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]です。 このページを使用すると、サブスクライバーがこのパブリッシャー上のパブリケーションからデータを受け取ることができます。 サブスクライバーがこのパブリッシャーからデータを受け取るだけでは、このパブリッシャー上のパブリケーションへのサブスクリプションは作成できません。 サブスクリプションを作成するには、サブスクリプションの新規作成ウィザードを使用します。  
  
## オプション  
 **[サブスクライバー]**  
  **サブスクライバー** このパブリッシャー上のパブリケーションからデータを受信するよう設定されているサブスクライバーがプロパティ グリッドに表示します。 [プロパティ] ボタンをクリックして (**...**) 表示し、追加のプロパティを設定するサブスクライバーの横にあります。  
  
 **[追加]**  
 クリックして **追加** サブスクライバーの追加] をクリックして **[SQL Server サブスクライバー** または **非 SQL Server サブスクライバーの追加**します。
