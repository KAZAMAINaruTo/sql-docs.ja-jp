# MSSQL_REPL-2147200941
    
## メッセージの詳細  
  
|||  
|-|-|  
|製品名|SQL Server|  
|イベント ID|-2147200941|  
|イベント ソース|MSSQLServer|  
|シンボル名||  
|メッセージ テキスト|マージ プロセスで、サブスクライバーのパーティションの検証式を評価しているときに、不一致を検出しました。 サブスクリプションを再初期化することにより、問題を解決できます。|  
  
## 説明  
 マージ プロセスで、サブスクライバーのパーティションの検証式を評価しているときに、不一致を検出しました。  
  
## ユーザーの操作  
 サブスクライバーを再初期化。
