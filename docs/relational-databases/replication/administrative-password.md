# 管理パスワード
  場合に、上、 **ディストリビューター** ページこのウィザードのこのパブリッシャーのリモート ディストリビューターを選択した、パブリッシャーとディストリビューターを使用して、間、接続のレプリケーションがのパスワードを入力する必要があります、 **distributor_admin** ログインします。 入力するパスワードは、ディストリビューションの構成ウィザードの **[ディストリビューター パスワード]** ページまたは **[ディストリビューターのプロパティ]** ダイアログ ボックスの **[パブリッシャー]** ページで指定されたパスワードと一致する必要があります。  
  
## オプション  
 **Password**  
 パブリッシャーとリモート ディストリビューターの間の接続のパスワードを入力します。  
  
 **[パスワードの確認入力]**  
 正しく入力されたことを確認するために、パスワードを再入力します。
