# [サブスクライバーのプロパティ]
   **サブスクライバー プロパティ** ] ダイアログ ボックスにはバージョンを実行しているサブスクライバーに関連する情報が含まれています [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] する前に [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]します。  
  
## オプション  
 **[サブスクライバーへのエージェント接続]**  
 ディストリビューターからサブスクライバーに、ディストリビューション エージェントとマージ エージェントを接続する際のコンテキストです。これは [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] より前のバージョンにのみ適用されます。  
  
 選択 **エージェント プロセス アカウントを借用** のコンテキストを使用してサブスクライバーへ接続する、 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] エージェントがディストリビューターでは、アカウントまたは指定 **SQL Server 認証**, の値を入力および **ログイン** と **パスワード**です。 [!INCLUDE[msCoName](../../includes/msconame-md.md)] 選択することをお勧め **エージェント プロセス アカウントを借用**します。  
  
  [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] とそれ以降のバージョンでは、接続の情報がサブスクリプションごとにサブスクリプションの新規作成ウィザードで指定されてで変更できます、 **サブスクリプションのプロパティ** ] ダイアログ ボックス。  
  
 **[既定のエージェントのスケジュール]**  
 バージョンを実行しているサブスクライバーのサブスクリプションの新規作成ウィザードで使用される既定のスケジュール [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] する前に [!INCLUDE[ssVersion2000](../../includes/ssversion2000-md.md)]します。  
  
 **その他**  
 サブスクライバーとサブスクライバーの種類に関する情報が含まれています。
