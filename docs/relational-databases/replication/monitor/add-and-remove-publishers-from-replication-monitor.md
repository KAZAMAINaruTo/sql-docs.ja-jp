# レプリケーション モニターのパブリッシャーの追加および削除
  パブリッシャーになっているサーバーからレプリケーション モニターを起動すると、そのサーバーは自動的にモニターに追加されます。 パブリッシャーを追加できる、 **[パブリッシャーの追加** ] ダイアログ ボックス。 追加したパブリッシャーは、モニターの左ペインのグループの中に表示されます。  **マイ パブリッシャー** グループは既定では、含まれていますが、1 つまたは複数のレプリケーション トポロジを管理する新しいグループを作成することができます。 レプリケーション モニターの起動方法については、次を参照してください。 [レプリケーション モニターを起動](../../../relational-databases/replication/monitor/start-the-replication-monitor.md)です。  
  
### SQL Server パブリッシャーを追加するには  
  
1.  右クリックし、 **レプリケーション モニター** ノードまたは左側のウィンドウでノードをグループ化し、をクリックし、パブリッシャー **[パブリッシャーの追加**します。  
  
2.   **[パブリッシャーの追加** ダイアログ ボックスで、をクリックして **追加**, 、] をクリックし、 **[SQL Server パブリッシャーの追加**します。  
  
3.   **サーバーへの接続** ] ダイアログ ボックスでは、パブリッシャーの名前を入力し、認証の種類を選択します。 選択した場合 **SQL Server 認証**, 、ログインとパスワードを入力します。 指定した資格情報はレプリケーション モニターによって保存され、今後このサーバーに接続するときに使用されます。 Windows アカウントまたは指定された SQL Server ログインのメンバーである必要があります、 **sysadmin** 固定サーバー ロールのメンバー、 **replmonitor** ディストリビューション データベースの固定データベース ロール。  
  
4.  **[接続]**をクリックします。 パブリッシャーでは、リモート ディストリビューターを使用している場合にディストリビューターに接続する適宜、 **サーバーへの接続** ] ダイアログ ボックス。 指定した資格情報はレプリケーション モニターによって保存され、今後このサーバーに接続するときに使用されます。 Windows アカウントまたは指定された SQL Server ログインのメンバーである必要があります、 **sysadmin** 固定サーバー ロールのメンバー、 **replmonitor** ディストリビューション データベースの固定データベース ロール。  
  
5.  パブリッシャーとディストリビューターの名前が表示される、 **次のパブリッシャーの監視を開始する** グリッドです。  
  
6.  パブリッシャーに対して更新や接続のオプションを指定するには、グリッドでパブリッシャーを選択し、必要に応じてオプションを変更します。 更新オプションの詳細については、次を参照してください。 [キャッシュ、更新、およびレプリケーション モニターのパフォーマンス](../../../relational-databases/replication/monitor/caching-refresh-and-replication-monitor-performance.md)します。  
  
7.  レプリケーション モニターでパブリッシャーを表示するグループを選択します。 新しいグループを作成する] をクリックして **新規グループ**, 、グループ名を入力する; でグループを選択、 **[このパブリッシャーを次のグループに表示** ] ボックスの一覧です。  
  
8.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
### Oracle パブリッシャーを追加するには  
  
1.  右クリックし、 **レプリケーション モニター** ノードまたは左側のウィンドウでノードをグループ化し、をクリックし、パブリッシャー **[パブリッシャーの追加**します。  
  
2.   **[パブリッシャーの追加** ダイアログ ボックスで、をクリックして **追加**, 、順にクリック **[Oracle パブリッシャーの**です。  
  
3.   **サーバーへの接続** ] ダイアログ ボックスの名前を入力、 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] ディストリビューターが Oracle パブリッシャーに関連付けられているし、認証の種類を選択します。 選択した場合 **SQL Server 認証**, 、ログインとパスワードを入力します。 指定した資格情報はレプリケーション モニターによって保存され、今後このサーバーに接続するときに使用されます。 Windows アカウントまたは指定された SQL Server ログインのメンバーである必要があります、 **sysadmin** 固定サーバー ロールのメンバー、 **replmonitor** ディストリビューション データベースの固定データベース ロール。  
  
4.  **[接続]**をクリックします。  
  
5.  パブリッシャーとディストリビューターの名前が表示される、 **次のパブリッシャーの監視を開始する** グリッドです。  
  
6.  パブリッシャーに対して更新や接続のオプションを指定するには、グリッドでパブリッシャーを選択し、必要に応じてオプションを変更します。 更新オプションの詳細については、次を参照してください。 [キャッシュ、更新、およびレプリケーション モニターのパフォーマンス](../../../relational-databases/replication/monitor/caching-refresh-and-replication-monitor-performance.md)します。  
  
7.  レプリケーション モニターでパブリッシャーを表示するグループを選択します。 新しいグループを作成する] をクリックして **新規グループ**, 、グループ名を入力する; でグループを選択、 **[このパブリッシャーを次のグループに表示** ] ボックスの一覧です。  
  
8.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
### 同じディストリビューターを使用する 1 つ以上のパブリッシャーを追加するには  
  
1.  右クリックし、 **レプリケーション モニター** ノードまたは左側のウィンドウでノードをグループ化し、をクリックし、パブリッシャー **[パブリッシャーの追加**します。  
  
2.   **[パブリッシャーの追加** ダイアログ ボックスで、をクリックして **追加**, 、] をクリックし、 **ディストリビューターとそのパブリッシャーを追加指定**します。  
  
3.   **サーバーへの接続** ] ダイアログ ボックスでは、ディストリビューターの名前を入力し、認証の種類を選択します。 選択した場合 **SQL Server 認証**, 、ログインとパスワードを入力します。 指定した資格情報はレプリケーション モニターによって保存され、今後このサーバーに接続するときに使用されます。 Windows アカウントまたは指定された SQL Server ログインのメンバーである必要があります、 **sysadmin** 固定サーバー ロールのメンバー、 **replmonitor** ディストリビューション データベースの固定データベース ロール。  
  
4.  **[接続]**をクリックします。  
  
5.  ディストリビューターと各パブリッシャーの名前が表示される、 **次のパブリッシャーの監視を開始する** グリッドです。 既にレプリケーション モニターに追加されているパブリッシャーはグリッドに表示されません。  
  
6.  パブリッシャーに対して更新や接続のオプションを指定するには、グリッドでパブリッシャーを選択し、必要に応じてオプションを変更します。 更新オプションの詳細については、次を参照してください。 [キャッシュ、更新、およびレプリケーション モニターのパフォーマンス](../../../relational-databases/replication/monitor/caching-refresh-and-replication-monitor-performance.md)します。  
  
7.  レプリケーション モニターでパブリッシャーを表示するグループを選択します。 新しいグループを作成する] をクリックして **新規グループ**, 、グループ名を入力する; でグループを選択、 **[このパブリッシャーを次のグループに表示** ] ボックスの一覧です。  
  
8.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
### パブリッシャーおよびパブリッシャー グループの設定を変更するには  
  
1.  左ペインでパブリッシャーを右クリックし、 **パブリッシャーの設定**です。  
  
2.  何も変更で、 **パブリッシャーの設定** ] ダイアログ ボックス。  
  
    -   レプリケーション モニターを使用してサーバーに接続する資格情報を変更するにはクリックして **パブリッシャー接続** または **ディストリビューター接続**, で資格情報を入力し、 **サーバーへの接続** ] ダイアログ ボックス。  
  
    -   パブリッシャーを選択、発行元を別の 1 つのグループに移動するに、 **次のパブリッシャーの監視を開始する** グリッドで新しいグループを選択し、 **[このパブリッシャーを次のグループに表示** ] ボックスの一覧です。  
  
3.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
### レプリケーション モニターからパブリッシャーを削除するには  
  
1.  左ペインでパブリッシャーを右クリックします。  
  
2.  **[削除]**をクリックします。  
  
### レプリケーション モニターにパブリッシャー グループを追加するには  
  
1.  パブリッシャー グループは、パブリッシャーを追加するとき、またはパブリッシャーの設定を変更するときにしか作成できません。 詳細については、パブリッシャーを追加する手順を参照してください。  
  
### レプリケーション モニターからパブリッシャー グループを削除するには  
  
1.  すべてのパブリッシャーを別のグループに移動するか、レプリケーション モニターから削除します。 詳細については、このトピックの上記の手順を参照してください。  
  
2.  パブリッシャー グループを右クリックし、をクリックし、 **削除**します。
