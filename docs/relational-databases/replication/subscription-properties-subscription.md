# <a name="subscription-properties---ltsubscriptiongt"></a>サブスクリプションのプロパティ - &lt;Subscription&gt;
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  ここでは **[サブスクリプションのプロパティ]** ダイアログ ボックスに関する次の情報について紹介します。  
  
-   「[サブスクリプションのプロパティ - サブスクライバー](../../relational-databases/replication/subscription-properties-subscriber.md)」ではサブスクライバーで使用できるサブスクリプションのプロパティについて説明します (プル サブスクリプションのみ)。  
  
-   「[Subscription Properties - Publisher](../../relational-databases/replication/subscription-properties-publisher.md) 」ではパブリッシャーで使用できるサブスクリプションのプロパティについて説明します (プル サブスクリプションおよびプッシュ サブスクリプションのみ)。
