# レプリケーション トポロジのセキュリティ保護
  レプリケーションに必要なセキュリティ アカウント、ログイン、およびパスワードは、レプリケーションの構成時とパブリケーションおよびサブスクリプションの作成時に指定します。 これらのセキュリティ設定は、[!INCLUDE[ssManStudioFull](../../../includes/ssmanstudiofull-md.md)]、レプリケーション ストアド プロシージャ、およびレプリケーション管理オブジェクト (RMO) を使用して、後から変更することもできます。 次のトピックでは、レプリケーションのセキュリティ設定を変更する方法について説明します。
