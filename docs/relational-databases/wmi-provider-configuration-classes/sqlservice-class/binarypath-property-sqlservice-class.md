## <a name="syntax"></a>構文  
  
```  
  
object.BinaryPath [= value]  
```  
  
## <a name="parts"></a>要素

 サービスを表す [SqlService クラス](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md) オブジェクト。
