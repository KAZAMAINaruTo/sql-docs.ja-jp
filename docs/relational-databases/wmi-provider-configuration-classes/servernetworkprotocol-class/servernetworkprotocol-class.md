[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [ServerNetworkProtocol クラス](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md)ネットワークのインスタンスにインストールされているプロトコルを表します[!INCLUDE[msCoName](../../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]ネットワーク経由でクライアントとの通信を有効にします。

-   ネットワーク プロトコルのプロパティおよび IP アドレスの参照。  
  
-   サーバー ネットワーク プロトコルの有効化および無効化。
