## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|SQL Server|  
|イベント ID|3260|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名|DMPIO_BUFFER_OVERFLOW|  
|メッセージ テキスト|内部バッファーがいっぱいになりました。|

操作を再試行してください。 同じエラーが再度発生した場合は、製品サポート担当者にご連絡ください。
