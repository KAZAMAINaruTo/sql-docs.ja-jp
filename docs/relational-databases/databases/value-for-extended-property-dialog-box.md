# 拡張プロパティの値ダイアログ ボックス
  **[*\<プロパティ名>* の値]** ダイアログ ボックスを使用すると、値を入力または表示できます。 これは、複数の場所から開くことができる共通ダイアログ ボックスです。  
  
## UI 要素の一覧  
 **[拡張プロパティ名]**  
 表示または設定される拡張プロパティの名前です。  
  
 **[拡張プロパティ値]**  
 拡張プロパティの値を入力または変更します。
