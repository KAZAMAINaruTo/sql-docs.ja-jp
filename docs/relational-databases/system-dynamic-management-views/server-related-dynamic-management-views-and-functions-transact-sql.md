# <a name="server-related-dynamic-management-views-and-functions-transact-sql"></a>サーバー関連の動的管理ビューおよび関数 (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  ここでは、ホスト サーバーにインストールされている SQL Server、フルテキスト、および SQL Server エージェントのサービスに関連する次の動的管理ビューについて説明します。 これらのビューを使用してサービスのプロパティ情報を返すことができます。 これらのビューにも、構成、インストール、およびメモリ ダンプ ファイルの情報が含まれています。
