# <a name="lesson-2-5---summary---managing-data-in-a-hierarchical-table"></a>レッスン 2-5 - 階層テーブルでのデータの管理
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)] **hierarchyid** データ型を使用すると、階層データを反映するテーブルを簡単に作成できるようになり、深さ優先および幅優先のインデックスを作成してテーブルに効率的にアクセスできます。 データをクエリおよび操作する場合は、階層的な手法を使用します。
