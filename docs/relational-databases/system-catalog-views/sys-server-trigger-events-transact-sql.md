# <a name="sysservertriggerevents-transact-sql"></a>sys.server_trigger_events (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  サーバー レベル (同期) トリガーが起動されるイベントごとに 1 行のデータを格納します。  
  
|列名|データ型|Description|  
|-----------------|---------------|-----------------|  
|**継承された列**||すべての列を継承[sys.server_events](../../relational-databases/system-catalog-views/sys-server-events-transact-sql.md)です。|  
|**is_first**|**bit**|トリガーはこのイベントに対して最初に起動されます。|  
|**is_last**|**bit**|トリガーはこのイベントに対して最後に起動されます。|
