# <a name="configuring-iis-for-the-microsoft-drivers-for-php-for-sql-server"></a>IIS の構成で Microsoft Drivers for PHP for SQL Server
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

このトピックに関するリソースへのリンクを提供する、[インターネット インフォメーション サービス (IIS) Web サイト](https://www.iis.net/)PHP アプリケーションをホストする IIS の構成に関連します。 ここに示すリソースは、IIS での FastCGI の使用に固有のものです。 FastCGI は、アプリケーション フレームワークの Common Gateway Interface (CGI) の実行可能ファイルが Web サーバーとやり取りできる標準プロトコルです。 FastCGI は、FastCGI は、複数の要求の CGI プロセスを再利用することで、標準の CGI プロトコルとは異なります。
