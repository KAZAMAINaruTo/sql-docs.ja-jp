# <a name="download-the-microsoft-drivers-for-php-for-sql-server"></a>For PHP for SQL Server、Microsoft ドライバーをダウンロードします。

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

## <a name="windows"></a>Windows

次のバージョンの Windows での PHP ドライバーのダウンロードのとおりです。

- [Microsoft Drivers 5.2 for PHP for SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=56729)
- [Microsoft Drivers 4.3 for PHP for SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=55642)
- [Microsoft ドライバー 4.0、3.2、3.1、3.0 for PHP for SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=20098)

## <a name="linux-and-macos"></a>Linux および macOS

PHP のドライバーは、簡単にダウンロードを使用してインストール PECL Linux や macOS にします。 参照してください、 [Linux や macOS インストール チュートリアル](installation-tutorial-linux-mac.md)詳細についてはします。 Linux および macOS での PHP のドライバーを手動でインストールする必要がある場合、次のバージョンはダウンロードできます。

- [Microsoft Drivers 5.2 for PHP for SQL Server](https://github.com/Microsoft/msphpsql/releases/tag/v5.2.0)
- [Microsoft Drivers 4.3 for PHP for SQL Server](https://github.com/Microsoft/msphpsql/releases/tag/v4.3.0)
