# <a name="pdostatement-class"></a>PDOStatement クラス
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

PDOStatement クラスは、ステートメントとステートメントの結果を表します。  
  
## <a name="syntax"></a>構文  
  
```  
  
PDOStatement {}  
```
