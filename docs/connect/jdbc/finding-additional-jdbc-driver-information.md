# <a name="finding-additional-jdbc-driver-information"></a>JDBC ドライバーの追加情報の検索
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  詳細については、[!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)]と[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]開発は、一般に、次のオンライン リソースを参照してください。  
  
## <a name="remarks"></a>解説  
  
|リソース|Description|  
|--------------|-----------------|  
|[JDBC Driver for SQL Server の GitHub リポジトリ](https://github.com/microsoft/mssql-jdbc)|このリポジトリには、SQL Server 用 JDBC Driver のソース コードが含まれています。 ファイルの問題には、このサイトを使用し、ドライバーに直接参加します。|
|[データ アクセスおよびストレージ デベロッパー センター](http://go.microsoft.com/fwlink?linkid=4173)|このサイトでは、ドキュメント、技術文書、サンプル コード、およびその他のリソースにすべてのデータ アクセス テクノロジ[!INCLUDE[msCoName](../../includes/msconame_md.md)]です。|  
|[SQL Server データ アクセス フォーラム](http://go.microsoft.com/fwlink/?LinkId=70651)|このサイトへのデータ アクセスに関するコミュニティ フォーラムを果たす[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]を使用して[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]Native Client、OLE DB、ODBC、ADO、MDAC、JDBC、または SOAP/HTTP です。|  
|[JDBC ブログ](http://go.microsoft.com/fwlink/?LinkId=124746)|このブログの使用に関する情報を提供する、[!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)]いて、JDBC ドライバー製品チームのメンバーと直接対話できます。|  
|[Microsoft Connect](http://go.microsoft.com/fwlink/?LinkID=116905)|このサイトでは、顧客が送信したドライバーの問題を検索でき、自分でも送信することができます (このサイトは英語の場合があります)。|
