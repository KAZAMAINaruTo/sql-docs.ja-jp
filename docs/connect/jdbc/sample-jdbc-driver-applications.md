# <a name="sample-jdbc-driver-applications"></a>サンプル JDBC Driver アプリケーション

 すべてのサンプル アプリケーションは、ローカル コンピューターでコンパイルおよび実行することができる *.java コード ファイルに含まれ、これらは次の場所の各サブフォルダーに格納されています。
