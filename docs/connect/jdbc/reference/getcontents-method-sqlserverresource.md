# <a name="getcontents-method-sqlserverresource"></a>getContents メソッド (SQLServerResource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  エラー文字列の情報を取得します。 このメソッドは内部でのみ使用されます。  
  
## <a name="syntax"></a>構文  
  
```  
  
protected java.lang.Object[][] getContents()  
```  
  
## <a name="return-value"></a>戻り値  
 多次元配列**オブジェクト**値。
