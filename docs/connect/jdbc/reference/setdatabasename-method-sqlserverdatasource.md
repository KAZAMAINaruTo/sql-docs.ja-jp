# <a name="setdatabasename-method-sqlserverdatasource"></a>setDatabaseName メソッド (SQLServerDataSource)

  接続するデータベース名を設定します。  
  
## <a name="syntax"></a>構文  
  
```  
  
public void setDatabaseName(java.lang.String databaseName)  
```  
  
#### <a name="parameters"></a>パラメーター  
 *databaseName*  
  
 A**文字列**データベース名を格納しています。
