# <a name="getmultisubnetfailover-method-sqlserverdatasource"></a>getMultiSubnetFailover メソッド (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  値を返します、 **multiSubnetFailover**接続プロパティです。  
  
## <a name="syntax"></a>構文  
  
```vb  
public boolean getMultiSubnetFailover();  
```  
  
## <a name="return-value"></a>戻り値  
 接続プロパティの現在の設定に応じて、true または false を返します。  
  
## <a name="remarks"></a>解説  
 詳細については、 **multiSubnetFailover**接続プロパティを参照してください[接続プロパティの設定](../../../connect/jdbc/setting-the-connection-properties.md)です。
