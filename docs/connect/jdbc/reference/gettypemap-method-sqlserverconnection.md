# <a name="gettypemap-method-sqlserverconnection"></a>getTypeMap メソッド (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  これに関連付けられているマップのオブジェクトを取得します[SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md)オブジェクト。  
  
> [!NOTE]  
>  このメソッドは現在サポートされていません、[!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]です。  
  
## <a name="syntax"></a>構文  
  
```  
  
public java.util.Map getTypeMap()  
```  
  
## <a name="return-value"></a>戻り値  
 マップ オブジェクトです。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>解説  
 この getTypeMap メソッドは、java.sql.Connection インターフェイスの getTypeMap メソッドによって指定されます。
