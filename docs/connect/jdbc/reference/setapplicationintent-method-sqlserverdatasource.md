# <a name="setapplicationintent-method-sqlserverdatasource"></a>setApplicationIntent メソッド (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  値を設定、 **applicationIntent**接続プロパティです。  
  
## <a name="syntax"></a>構文  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>パラメーター  
 *applicationIntent*  
  
 新しい値、 **applicationIntent**接続プロパティです。  
  
## <a name="remarks"></a>解説  
 詳細については、 **applicationIntent**接続プロパティを参照してください[接続プロパティの設定](../../../connect/jdbc/setting-the-connection-properties.md)です。
