# <a name="getdouble-method-int"></a>getDouble (int) メソッド

  として指定されたパラメーターの値を取得、**二重**java プログラミング言語のパラメーターのインデックスを指定します。  
  
## <a name="syntax"></a>構文  
  
```  
  
public double getDouble(int index)  
```  
  
#### <a name="parameters"></a>パラメーター  
 *インデックス*  
  
 **Int**パラメーター インデックスを示すです。

 A**二重**値。
