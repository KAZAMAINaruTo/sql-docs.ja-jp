# <a name="clearwarnings-method-sqlserverstatement"></a>clearWarnings メソッド (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  これで報告されたすべての警告をクリア[SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md)オブジェクト。  
  
## <a name="syntax"></a>構文  
  
```  
  
public final void clearWarnings()  
```  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>解説  
 この clearWarnings メソッドは、clearWarnings、java.sql.Statement インターフェイスのメソッドでによって指定されます。
