# <a name="datetimeoffset-class"></a>DateTimeOffset クラス
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  この型は、microsoft.sql.DateTimeOffset 型のオブジェクトを表しますあり、Java の型に対応する、 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] **datetimeoffset**データ型。  
  
 **パッケージ:** microsoft.sql  
  
 **拡張:** java.lang.Object  
  
 **実装:** java.lang.Comparable\<DateTimeOffset >、java.io.Serializable  
  
## <a name="syntax"></a>構文  
  
```  
  
public final class DateTimeOffset  
```
