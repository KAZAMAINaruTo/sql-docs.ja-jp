# <a name="close-method-sqlserverpooledconnection"></a>close メソッド (SQLServerPooledConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  物理的な接続を閉じますこの[SQLServerPooledConnection](../../../connect/jdbc/reference/sqlserverpooledconnection-class.md)オブジェクトが表すです。  
  
## <a name="syntax"></a>構文  
  
```  
  
public void close()  
```  
  
## <a name="exceptions"></a>例外  
 java.sql.SQLException  
  
## <a name="remarks"></a>解説  
 このメソッドは、閉じる、javax.sql.PooledConnection インターフェイスのメソッドでによって指定されます。
