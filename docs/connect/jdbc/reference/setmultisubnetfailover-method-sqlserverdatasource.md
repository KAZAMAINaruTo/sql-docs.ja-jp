# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>setMultiSubnetFailover メソッド (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  値を設定、 **multiSubnetFailover**接続プロパティです。  
  
## <a name="syntax"></a>構文  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>パラメーター  
 *multiSubnetFailover*  
  
 新しい値、 **multiSubnetFailover**接続プロパティです。  
  
## <a name="remarks"></a>解説  
 詳細については、 **multiSubnetFailover**接続プロパティを参照してください[接続プロパティの設定](../../../connect/jdbc/setting-the-connection-properties.md)です。
