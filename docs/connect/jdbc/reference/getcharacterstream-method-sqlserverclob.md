# <a name="getcharacterstream-method-sqlserverclob"></a>getCharacterStream メソッド (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  取得、 **Clob** java.io.Reader オブジェクトとして java.sql.Clob オブジェクトによって指定された値。  
  
## <a name="overload-list"></a>オーバーロードの一覧  
  
|名前|Description|  
|----------|-----------------|  
|[getCharacterStream メソッド&#40;&#41;](../../../connect/jdbc/reference/getcharacterstream-method.md)|Clob データを、java.io.Reader オブジェクトまたは文字のストリームとして返します。|  
|[getCharacterStream メソッド&#40;long long&#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Java.io.Reader オブジェクトまたは指定した位置と長さを文字のストリームとしては、Clob データを返します。|
