# <a name="addbatch-method-"></a>addBatch () メソッド
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

## <a name="syntax"></a>構文  
  
```  
  
public final void addBatch()  
```  
  
## <a name="exceptions"></a>例外

## <a name="remarks"></a>解説
