# <a name="getclob-method-sqlservercallablestatement"></a>getClob メソッド (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  指定された JDBC BLOB パラメーターの値を Java プログラミング言語では、Clob オブジェクトとして取得します。  
  
## <a name="overload-list"></a>オーバーロードの一覧  
  
|名前|Description|  
|----------|-----------------|  
|[getClob (int)](../../../connect/jdbc/reference/getclob-method-int.md)|指定された JDBC BLOB パラメーターの値を Java プログラミング言語のパラメーター インデックス Clob オブジェクトとして取得します。|  
|[getClob (java.lang.String)](../../../connect/jdbc/reference/getclob-method-java-lang-string.md)|パラメーター名を指定、Java プログラミング言語の Clob オブジェクトとして指定された JDBC BLOB パラメーターの値を取得します。|
