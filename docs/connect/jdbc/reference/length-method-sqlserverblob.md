# <a name="length-method-sqlserverblob"></a>length メソッド (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  BLOB オブジェクト内のバイト数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>戻り値  
 A**長い**のバイト数を指定する値。
