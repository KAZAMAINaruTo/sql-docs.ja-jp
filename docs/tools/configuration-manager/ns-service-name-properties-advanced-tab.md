# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;サービス名&gt;プロパティ ([詳細設定] タブ)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)]という形式の名前のサービスとして実装**NS$***< user_configured name >*です。 カスタム プロパティが定義されていれば、そのプロパティと値がこのタブに表示されます。 このタブにプロパティが表示されない場合もあります。
