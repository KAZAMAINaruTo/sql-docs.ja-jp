# <a name="name-element-for-schema-dta"></a>Schema の Name 要素 (DTA)
  スキーマの名前が入ります。  
  
## <a name="syntax"></a>構文  
  
```  
  
<Database>  
...code removed here  
    <Schema>  
        <Name>...</Name>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|**データ型と長さ**|**string**、1 ～ 255 文字。|  
|**既定値**|[なし] :|  
|**個数**|**Schema** 要素につき 1 回の出現が必要です。|  
  
## <a name="element-relationships"></a>要素の関係

|**子要素**|[なし] :|
