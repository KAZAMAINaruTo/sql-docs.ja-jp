# Database の Name 要素 (DTA)
  チューニングするデータベースの名前を指定します。  
  
## 構文  
  
```  
  
<Server>  
    <Database>  
        <Name>...</Name>  
```  
  
## 要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|**データ型と長さ**|**string**、長さは無制限です。|  
|**既定値**|[なし] :|  
|**個数**|**Database** 要素につき 1 個が必要です。|  
  
## 要素の関係  
  
|リレーションシップ|要素|  
|------------------|--------------|  
|**親要素**|[Server の Database 要素 &#40;DTA&#41;](../../tools/dta/database-element-for-server-dta.md)|  
|**子要素**|[なし] :|
