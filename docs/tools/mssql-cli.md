# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>SQL Server の mssql cli コマンド ライン クエリ ツール
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

mssql cli は、SQL Server を照会するための対話的なコマンド ライン ツール、Windows、macOS、または Linux をインストールします。

## <a name="install-mssql-cli"></a>Mssql cli をインストールします

詳細なインストール手順については、次を参照してください。、[インストール ガイド](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md)、または pip がわかっている場合は、次のコマンドを実行してインストールします。

```$ pip install mssql-cli```
