# <a name="proxy-account-properties-references-tab"></a>[プロキシ アカウントのプロパティ] ([参照設定] タブ)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] この読み取り専用ページには、[!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] エージェントのプロキシ アカウントを参照するジョブ ステップが一覧表示されます。  
  
## <a name="options"></a>オプション  
**ジョブ名**  
このプロキシ アカウントを使用するジョブの名前です。  
  
**[ジョブ ステップ]**  
このプロキシ アカウントを使用する、ジョブ内のジョブ ステップの名前です。  
  
**[サブシステム]**  
このプロキシ アカウントで使用されるサブシステムです。
