# <a name="delete-operator"></a>Delete 操作
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] このページを使用すると、オペレーターを削除できます。  
  
## <a name="options"></a>および  
**[削除されるオブジェクト]**  
削除されるオペレーターを表示します。  
  
**[再割り当てするオペレーター]**  
削除されるオペレーターに通知を再割り当てします。  
  
**[プロパティ]**  
通知を再割り当てする対象オペレーターのプロパティを表示します。
