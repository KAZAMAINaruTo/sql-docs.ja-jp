# <a name="getlogwriter-method-sqlserverdatasource"></a>getLogWriter メソッド (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  このメソッドは、内部でのみ使用されます。 ログ記録の詳細については、次を参照してください。[ドライバー操作のトレース](../../../connect/jdbc/tracing-driver-operation.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>戻り値  
 PrintWriter オブジェクトです。  
  
## <a name="remarks"></a>解説  
 この getLogWriter メソッドは、javax.sql.DataSource インターフェイスの getLogWriter メソッドによって指定されます。
