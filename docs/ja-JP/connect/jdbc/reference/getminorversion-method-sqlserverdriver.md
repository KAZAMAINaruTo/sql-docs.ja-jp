# <a name="getminorversion-method-sqlserverdriver"></a>getMinorVersion メソッド (SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  マイナー バージョン番号を返します、[!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]です。  
  
## <a name="syntax"></a>構文  
  
```  
  
public int getMinorVersion()  
```  
  
## <a name="return-value"></a>戻り値  
 **Int** JDBC ドライバーのマイナー バージョン番号を格納しています。  
  
## <a name="remarks"></a>解説  
 この getMinorVersion メソッドは、java.sql.Driver インターフェイスの getMinorVersion メソッドによって指定されます。
