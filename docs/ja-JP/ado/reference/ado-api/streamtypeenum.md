# <a name="streamtypeenum"></a>StreamTypeEnum
格納されたデータの種類を指定、[ストリーム](../../../ado/reference/ado-api/stream-object-ado.md)オブジェクト。  
  
|定数|値|Description|  
|--------------|-----------|-----------------|  
|**adTypeBinary**|1|バイナリ データを示します。|  
|**adTypeText**|2|既定値です。 文字で指定されたセットでは、テキスト データを示す[Charset](../../../ado/reference/ado-api/charset-property-ado.md)です。|
