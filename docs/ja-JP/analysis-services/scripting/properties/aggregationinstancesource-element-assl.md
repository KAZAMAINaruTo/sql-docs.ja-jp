# <a name="aggregationinstancesource-element-assl"></a>AggregationInstanceSource 要素 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<Partition>  
   ...  
   <AggregationInstanceSource xsi:type="DataSourceViewBinding">...</AggregationInstanceSource>  
   ...  
</Partition>  
```

|--------------------|-----------------|
