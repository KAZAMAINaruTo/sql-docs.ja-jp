# <a name="customrollup-element-xmla"></a>CustomRollup 要素 (XMLA)

## <a name="syntax"></a>構文  
  
```xml  
  
<Attribute>  
   ...  
   <CustomRollup>...</CustomRollup>  
   ...  
</Attribute>  
```  
  
## <a name="element-characteristics"></a>要素の特性

|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|
