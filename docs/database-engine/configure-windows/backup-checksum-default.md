# バックアップ チェックサムの既定
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  バックアップと復元中にバックアップ チェックサムを有効または無効にするために、バックアップ チェックサムの既定の設定を使用します。  
  
 次の表に、このオプションで使用可能な値を示します。  
  
|||  
|-|-|  
|**値**|**意味**|  
|0|無効。 これが既定の設定です。|  
|1|有効|  
  
 この設定はすぐに有効になります。
