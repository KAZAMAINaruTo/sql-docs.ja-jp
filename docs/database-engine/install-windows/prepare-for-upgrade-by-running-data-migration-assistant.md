# アップグレード アドバイザーの実行によるアップグレードの準備
  Microsoft Data Migration Assistant をダウンロードして実行し、OLTP データベースをアップグレードして [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] の新機能を採用する準備をします。  
  
## <a name="download-and-install-data-migration-assistant"></a>Data Migration Assistant をダウンロードしてインストールする  
 Data Migration Assistant を[ここ](http://go.microsoft.com/fwlink/?LinkID=613421)からダウンロードしてインストールします。 このツールは、 [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] インストール メディアには含まれていません。
