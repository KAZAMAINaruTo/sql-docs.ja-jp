# <a name="wide-world-importers-data-warehousing-and-analytics-database"></a>Wide World importers 社のデータ ウェアハウスと分析データベース

Wide World importers 社のデータ ウェアハウスと分析データベースのドキュメントです。

**WideWorldImportersDW インストールと構成**

インストールと OLAP の構成手順については、WideWorldImportersDW をデータベースです。

**WideWorldImportersDW OLAP データベースのカタログ**

**WideWorldImporters ETL ワークフロー**

トランザクション データベース WideWorldImporters からのデータをデータ ウェアハウス WideWorldImportersDW に移行する ETL (抽出、変換、読み込み) プロセスのワークフローです。

**SQL Server の機能と機能の使用を WideWorldImportersDW**

WideWorldImportersDW が分析処理に対して SQL Server の機能を活用する方法について説明します。

**WideWorldImportersDW OLAP のサンプル クエリ**

サンプルの分析クエリが WideWorldImportersDW データベースを活用することです。
