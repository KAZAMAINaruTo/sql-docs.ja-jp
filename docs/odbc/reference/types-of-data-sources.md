# <a name="types-of-data-sources"></a>データ ソースの種類
データ ソースの 2 種類があります。 コンピューターのデータ ソースとファイル データ ソース。 両方は、データのソースに関する同様の情報が含まれていますが、この情報の格納方法が異なります。 これらの違いにより若干異なる方法で使用されます。
