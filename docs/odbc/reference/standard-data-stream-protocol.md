# <a name="standard-data-stream-protocol"></a>標準のデータ ストリーム プロトコル
標準的なデータ ストリーム プロトコルとは、異種の Dbms でデータにアクセスする方法の 1 つです。 標準的なデータ ストリーム プロトコルが既に存在する実際には、: ANSI/ISO リモート データベースへのアクセス (RDA) 標準 ([ISO/IEC 9579:2000](http://www.iso.org/iso/catalogue_detail.htm?csnumber=30615)) と、IBM 分散型リレーショナル データベース アーキテクチャ (DRDA) ([ http://www.netstoreusa.com/cbbooks/007/0070351384.shtml](http://www.netstoreusa.com/cbbooks/007/0070351384.shtml)). 両方のシステムでは、promise を表示、どちらも広く実装今日されます。
