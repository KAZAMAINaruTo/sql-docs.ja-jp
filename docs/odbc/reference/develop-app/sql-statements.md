# <a name="sql-statements"></a>SQL ステートメント
ODBC アプリケーションでは、SQL ステートメントを実行することによってほとんどすべてのデータベースへのアクセスを実行します。 これらのステートメントの形式:、ハードコーディングまたは実行時に、相互運用可能なデータやデータ ソースの特定などに構築された、アプリケーションのニーズによって異なります。
