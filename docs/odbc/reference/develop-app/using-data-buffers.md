# <a name="using-data-buffers"></a>データ バッファーを使用します。
データ バッファーが 3 つの情報が記載されている: その種類、アドレス、およびバイトの長さ。 関数は、これらの種類の情報のうち 1 つしは理解していないこと、ときに使用するアプリケーションに渡すこと引数を持ちます。
