# <a name="using-scrollable-cursors"></a>スクロール可能なカーソルを使用します。
スクロール可能なカーソルを使用するには、これら 3 つの手順が必要です。  
  
1.  カーソル機能が決まります。  
  
2.  カーソルを設定します。  
  
3.  行のスクロールおよびフェッチします。
