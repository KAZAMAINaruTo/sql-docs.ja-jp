# <a name="obtaining-descriptor-handles"></a>処理記述子を取得します。
アプリケーションへの呼び出しの出力引数としての明示的に割り当てられた記述子ハンドルを取得する**SQLAllocHandle**です。 呼び出すことによって暗黙的に割り当てられた記述子ハンドルを取得**SQLGetStmtAttr**です。
