# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Visual Studio Analyzer のサポートが以降 (Visual Studio Analyzer のみに含まれていました Visual Studio の古いバージョンです。) Windows 8 では削除されました。 トラブルシューティング メカニズムの代わりの BID トレースを使用します。  
  
 Microsoft® Visual Studio™ アナライザーは、ODBC アプリケーションの概要を提供します。 開発者は評価、分析、および構造体、パフォーマンス、およびコンポーネントまたはコードの観点からではなく、アプリケーションの観点からの相互作用をデバッグします。 Visual Studio Analyzer によって収集された情報は、ある種のアプリケーションの 2 つのコンポーネント間の相互作用を表すイベントの形式です。 関数呼び出しとの接続、データベース クエリ、およびトランザクションなどの関数からのイベントを構成できます。
