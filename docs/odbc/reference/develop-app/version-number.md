# <a name="version-number"></a>バージョン番号
ODBC では、それぞれさまざまな機能のいくつかのバージョンがあります。 ODBC ドライバー マネージャーのバージョンと特定のドライバーをサポートして呼び出すことによって判断**SQLGetInfo** SQL_ODBC_VER と SQL_DRIVER_ODBC_VER オプションを使用します。
