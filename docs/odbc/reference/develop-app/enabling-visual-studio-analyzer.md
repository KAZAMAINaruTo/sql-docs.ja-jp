# <a name="enabling-visual-studio-analyzer"></a>Visual Studio Analyzer を有効にします。
> [!IMPORTANT]  
>  Visual Studio Analyzer のサポートが以降 (Visual Studio Analyzer のみに含まれていました Visual Studio の古いバージョンです。) Windows 8 では削除されました。 トラブルシューティング メカニズムの代わりの BID トレースを使用します。  
  
 Microsoft Visual Studio Analyzer は、すべての階層とシステム間で、概要を見て、アプリケーションのパフォーマンスを提供するアプリケーション分析ツールです。 これは、コンポーネント間の相互作用について説明します。 Visual Studio Analyzer を開始し、ODBC アドミニストレーターのトレース タブから停止できます。  
  
 ODBC で生成されるイベントを表示するには、次の手順に従います。  
  
1.  ODBC データ ソース アドミニストレーターにアクセスします。  
  
2.  クリックして、**トレース**タブです。  
  
3.  をクリックして**Visual Studio Analyzer のトレースを開始**です。  
  
4.  **[OK]**をクリックします。  
  
 Visual Studio Analyzer イベント世代まで**Visual Studio Analyzer のトレースを停止**が選択されています。
