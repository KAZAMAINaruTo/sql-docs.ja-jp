 Sqlext.h

 Msdadc.h  
  
 Sqlspi.h (ドライバーの開発に使用するを参照してください[ODBC サービス プロバイダー インターフェイス (SPI) 参照](../../../odbc/reference/syntax/odbc-service-provider-interface-spi-reference.md)詳細については) 以降、Windows 8 SDK で使用できます。
