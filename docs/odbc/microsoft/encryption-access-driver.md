# <a name="encryption-access-driver"></a>暗号化 (Access ドライバー)
(このコントロールは、使用可能な場合にのみ Version 4。*x*形式を選択します)。  
  
 4.0、Microsoft Jet データベース エンジンのバージョンのリリースにより、Access データベースを作成するときにデータベースの暗号化を許可します。 参照してください、 *Jet データベース エンジン プログラマーズ リファレンス*の詳細。
