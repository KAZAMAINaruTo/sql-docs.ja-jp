# <a name="date-arithmetic"></a>日付の算術演算子
ドライバーは、加算と減算の日付、時刻、またはタイムスタンプ列から integer をサポートします。 日付の列は、整数は、加算または減算する日数を指定します。 時間またはタイムスタンプ列の場合は、整数は、加算または減算する秒数を指定します。
