# <a name="where-clause-limitations"></a>WHERE 句の制限事項
WHERE 句内の句の最大数は、40 です。  
  
 LONGVARBINARY および LONGVARCHAR 列で、最大 255 文字のリテラルと比較することができますが、パラメーターを使用して比較することはできません。
