# <a name="where-clause-limitations"></a>ここで句の制限事項
WHERE 句内の句の最大数は 40 です。  
  
 LONGVARBINARY および LONGVARCHAR 列は、長さ、最大 255 文字のリテラルを比較することができますが、パラメーターを使用して比較することはできません。
