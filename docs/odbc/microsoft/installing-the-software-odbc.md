# <a name="installing-the-software-odbc"></a>インストール (ODBC)
> [!IMPORTANT]  
>  この機能は、Windows の将来のバージョンで削除されます。 新規の開発作業ではこの機能を使用しないようにし、現在この機能を使用しているアプリケーションは修正することを検討してください。 代わりに、Oracle によって提供される ODBC ドライバーを使用します。  
  
 ODBC Driver for Oracle は、データ アクセス コンポーネントのいずれかです。 ODBC データ ソース アドミニストレーターなど、他の ODBC コンポーネントに付属し、既にインストールされている必要があります。 ドライバーも見つかります「ドライバーおよびその他のダウンロード」www.microsoft.com のマイクロソフト製品サポート サービスのオンライン Web サイトでします。  
  
 独自のドキュメントに従って、ネットワーク ソフトウェアをインストールする必要があります。 ネットワーク ソフトウェアがサポートされている限り、oracle ODBC ドライバーによって特殊なインストールに関する注意点は必要ありません。  
  
 独自のドキュメントに従って、oracle ソフトウェアをインストールする必要があります。 ODBC Driver for Oracle、通常は不要特別なインストールに関する注意点ドライバー バージョンをサポートしている限りです。 ただし、製品の互換性を維持するには、最後に、ドライバーの最新バージョンであることを確認してください。 Oracle の ODBC ドライバーをインストールします。 Oracle では、ここで、投稿記事、特に、パッチ、Oracle サーバー製品とのサーバー製品に付属しているクライアント コンポーネントをパブリック FTP サイトを維持します。 これらの更新プログラムは、いくつかの Microsoft 製品およびテクノロジの正しく機能するため必要があります。 このサイトに関する詳細については、次を参照してください。 [Oracle ソフトウェアの修正プログラム](../../odbc/microsoft/oracle-software-patches.md)です。  
  
> [!CAUTION]  
>  MDAC/Windows DAC 経由での Oracle ソフトウェアをインストールすると、現在のバージョンの MDAC が上書きされます。 ODBC コンポーネントを使用して問題が発生した場合は、MDAC を再インストールします。
