# <a name="microsoft-odbc-desktop-database-drivers"></a>デスクトップのデータベースの Microsoft ODBC ドライバー
ODBC は、データベース アクセスの言語として構造化照会言語 (SQL) を使用する API です。 アプリケーションのソース コードに直接組み込まれている同じ ODBC ソース コードにさまざまなデータベース管理システム (Dbms) にアクセスできます。 Microsoft ODBC のデスクトップ データベース ドライバーと ODBC 対応アプリケーションのユーザーが開くには、クエリ、および ODBC インターフェイスを使用してデスクトップ データベースを更新します。  
  
 Microsoft ODBC のデスクトップ データベース ドライバーは、Microsoft Jet ベースの ODBC ドライバーのセットです。 3.0 以降のバージョンが Windows NT ワークステーションまたはサーバー バージョン 4.0、Windows 2000 Professional、または Windows 2000、Windows 95 以降の機能のみの 32 ビット ドライバーを含む Microsoft ODBC デスクトップ データベース用 Drivers 2.0 には、16 ビットおよび 32 ビットの両方のドライバーが含まれて、一方サーバー。 これらのドライバーは、次の種類のデータ ソースへのアクセスを提供します。  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   Paradox  
  
-   DBASE  
  
-   テキスト  
  
 参照してください[Visual FoxPro ODBC ドライバー](../../odbc/microsoft/visual-foxpro-odbc-driver.md)の Microsoft Visual FoxPro® ODBC Driver に関する詳細なドキュメントです。  
  
> [!NOTE]  
>  Lotus 1 2 3、Microsoft Exchange、HTML などの他のデータ ソースへのアクセスは、インストール可能な ISAM (IISAM) ドライバーでは有効です。 これらのドライバーの詳細については、「外部データにアクセスする」を参照してください、 *Microsoft Jet データベース エンジン プログラマーズ リファレンス*です。 ODBC デスクトップ データベース ドライバー 4.0 は、市販と EMS データ形式をサポートしていません。
