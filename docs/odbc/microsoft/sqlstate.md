# <a name="sqlstate"></a>SQLSTATE
データ ソースで発生したエラーの場合は、ODBC ドライバーは、返されたネイティブ エラーを適切な SQLSTATE にマップされます。 ドライバーまたは ODBC ドライバー、ドライバー マネージャーによって検出されたエラーまたはドライバー マネージャーが適切な SQLSTATE を生成します。
