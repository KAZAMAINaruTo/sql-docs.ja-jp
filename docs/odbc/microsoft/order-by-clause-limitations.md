# <a name="order-by-clause-limitations"></a>ORDER BY 句の制限事項
GROUP BY 句と ORDER BY 句が SELECT ステートメントが含まれる場合、ORDER BY 句は結果セット内の列のみまたは GROUP BY 句の式を含めることができます。
