# <a name="product-support"></a>製品のサポート
ODBC の製品サポートは、マイクロソフト製品サポート サービスによって提供されます。 多くの Microsoft 製品では、主要なコンポーネントとして ODBC を使用し、ODBC ドライバーを再配布もため、マイクロソフト製品サポート サービスは ODBC コンポーネントとして ODBC が組み込まれている各製品の一部であると見なします。 サポート ODBC がその製品のサポート契約に従って提供します。 詳細については、個々 の製品ドキュメントを参照してください。
