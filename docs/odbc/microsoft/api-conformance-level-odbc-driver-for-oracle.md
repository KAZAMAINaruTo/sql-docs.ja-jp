# <a name="api-conformance-level-odbc-driver-for-oracle"></a>API への準拠レベル (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  この機能は、Windows の将来のバージョンで削除されます。 新規の開発作業ではこの機能を使用しないようにし、現在この機能を使用しているアプリケーションは修正することを検討してください。 代わりに、Oracle によって提供される ODBC ドライバーを使用します。  
  
 ODBC Driver for Oracle は、[レベルの主要な API 関数](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md)と[レベル 1 の API 関数](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md)です。 ドライバーは、以下もサポートしています[レベル 2 の API 関数](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md):。  
  
-   [SQLBrowseConnect に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLDataSources に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLDescribeParam に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLExtendedFetch)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLForeignKeys に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLMoreResults に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLNativeSql に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLNumParams に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)  
  
-   [SQLPrimaryKeys に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)

-   [SQLProcedures に関するページ)](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)
