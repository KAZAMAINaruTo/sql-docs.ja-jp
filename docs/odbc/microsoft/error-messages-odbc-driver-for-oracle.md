# <a name="error-messages-odbc-driver-for-oracle"></a>エラー メッセージ (ODBC Driver for Oracle)
> [!IMPORTANT]  
>  この機能は、Windows の将来のバージョンで削除されます。 新規の開発作業ではこの機能を使用しないようにし、現在この機能を使用しているアプリケーションは修正することを検討してください。 代わりに、Oracle によって提供される ODBC ドライバーを使用します。  
  
 エラーが発生するときに、Microsoft ODBC Driver for Oracle は、SQLSTATE (ODBC エラー コード) とエラー メッセージを返します。 ドライバーは、ドライバーによって検出されたエラーと、Oracle サーバーにより返されたエラーの両方に、この情報を取得します。
