# <a name="sorting-limitations"></a>制限事項を並べ替え
GROUP BY 句、ORDER BY 句、SELECT DISTINCT ステートメントでは、または外部結合、並べ替えキーの最大長は 255 バイトを表します。並べ替え行内のすべての並べ替えキーの最大長は 65,500 バイトです。
