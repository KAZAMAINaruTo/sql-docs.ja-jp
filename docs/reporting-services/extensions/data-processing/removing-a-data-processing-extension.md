# <a name="removing-a-data-processing-extension"></a>データ処理拡張機能を削除します。
  削除するには、データ処理拡張機能を削除するだけ、**拡張子**構成ファイルからデータ処理拡張機能の要素。 レポート デザイナーだけでなくレポート サーバーのエントリを作成した場合は、削除、**拡張子**RSReportDesigner.config と RSReportServer.config ファイルからの要素。 構成情報を削除すると、そのコンポーネントではデータ処理拡張機能を使用できなくなります。
