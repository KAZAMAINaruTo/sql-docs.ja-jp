# <a name="how-to-find-the-product-key-for-sql-server-2017-reporting-services"></a>SQL Server 2017 Reporting Services のプロダクト キーを見つける方法

[!INCLUDE[ssrs-appliesto](../../includes/ssrs-appliesto.md)] [!INCLUDE[ssrs-appliesto-2017-and-later](../../includes/ssrs-appliesto-2017-and-later.md)] [!INCLUDE[ssrs-appliesto-not-pbirsi](../../includes/ssrs-appliesto-not-pbirs.md)]

運用環境でサーバーを設置できるように、SQL Server 2017 Reporting Services (SSRS) プロダクト キーを見つける方法について説明します。

プロダクト キーを見つけるには、まず、SQL Server 2017 をダウンロードし、セットアップを実行します。

1. SQL Server 2017 をダウンロードできるソース:

    - ボリューム ライセンス サービス センター
    - MSDN サブスクリプション
    - 小売 (Microsoft Store からダウンロード)

1. SQL Server 2017 セットアップを実行し、事前入力されているキーをコピーします。

    ![SQL Server 2017 プロダクト キーをコピーする](media/find-reporting-services-product-key-ssrs/ssrs-ss2017-copy-product-key.png)

1. [Reporting Services セットアップを実行し](install-reporting-services.md)、キーを貼り付けます。

     ![プロダクト キーを貼り付ける](media/find-reporting-services-product-key-ssrs/ssrs-ssrs2017-paste-product-key.png)

この手順は、SSRS 2017 を初めてインストールするときにだけ行います。 サービス更新では、キーを入力する必要がありません。
