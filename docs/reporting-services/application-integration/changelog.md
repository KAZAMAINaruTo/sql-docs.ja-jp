# <a name="changelog-for-the-report-viewer-controls"></a>レポート ビューアー コントロールの変更ログ

この変更ログは、WinForms と WebForms のレポート ビューアー コントロールのリリースを追跡します。

## <a name="15900148"></a>15.900.148
この更新プログラムの内容
 - パラメーターのないレポートが Server.LoadReportDefinition によって読み込まれないバグが修正されています

[WebForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Webforms/150.900.148) レポート ビューアー コントロール
 - RTL ページ (*direction: rtl;* css を使用してテキスト フローをを変更するページ) への埋め込みがサポートされました。
 - *IReportViewerMessages5* ローカリゼーション インターフェイスを使用した、印刷ダイアログ テキストのカスタマイズがサポートされました。
 - ユーザー補助のサポートが改善されました。

[WinForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Winforms/150.900.148) レポート ビューアー コントロール
 - アプリが高 DPI モードで実行されている場合の印刷機能が修正されました。
