# <a name="configurationsetting-property---virtualdirectoryreportmanager"></a>VirtualDirectoryReportManager ConfigurationSetting プロパティ
  レポート マネージャー アプリケーションの仮想ディレクトリを返します。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## <a name="property-values"></a>プロパティ値  
 仮想ディレクトリを表す **string** 。
