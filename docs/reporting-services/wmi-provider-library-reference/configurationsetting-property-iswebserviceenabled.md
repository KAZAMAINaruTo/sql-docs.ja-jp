# <a name="configurationsetting-property---iswebserviceenabled"></a>ConfigurationSetting プロパティ - IsWebServiceEnabled
  レポート サーバー Web サービスが有効かどうかを示します。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>プロパティ値  
 読み取り専用の **ブール値** です。 値 **true** は、レポート サーバー Web サービスが有効であることを示します。
