# <a name="configurationsetting-property---sendusingsmtpserver"></a>ConfigurationSetting プロパティ - SendUsingSMTPServer
  レポート サーバーの電子メール構成で **SendUsing** 要素を **true**に設定するかどうかを指定します。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim SendUsingSMTPServer As Boolean  
```  
  
```csharp  
public Boolean SendUsingSMTPServer;  
```  
  
## <a name="property-values"></a>プロパティ値  
 読み取り専用 **ブール値** オブジェクトです。
