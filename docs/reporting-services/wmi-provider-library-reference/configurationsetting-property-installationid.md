# <a name="configurationsetting-property---installationid"></a>ConfigurationSetting プロパティ - InstallationID
  このプロパティは、特定のレポート サーバー インスタンスの一意な識別子を返します。  
  
## <a name="syntax"></a>構文  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
