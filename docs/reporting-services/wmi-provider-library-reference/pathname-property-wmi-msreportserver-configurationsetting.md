# PathName プロパティ (WMI MSReportServer_ConfigurationSetting)
  レポート サーバー インスタンスのインストール パスを指定します。 読み取り専用です。  
  
## 構文  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## プロパティ値  
 インストール パスを表す **String** オブジェクト。
