# <a name="configurationsetting-property---windowsserviceidentityactual"></a>WindowsServiceIdentityActual ConfigurationSetting プロパティ
  レポート サーバー Windows サービスを実行している ID です。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim WindowsServiceIdentityActual As String  
```  
  
```csharp  
public string WindowsServiceIdentityActual;  
```  
  
## <a name="property-values"></a>プロパティ値  
 レポート サーバー Windows サービスを実行している ID を含む **String** オブジェクト。
