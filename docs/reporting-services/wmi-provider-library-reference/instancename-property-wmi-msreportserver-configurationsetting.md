# InstanceName プロパティ (WMI MSReportServer_ConfigurationSetting)
  特定のコンピューターのレポート サーバー インスタンスの名前を指定します。 読み取り専用です。  
  
## 構文  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## プロパティ値  
 レポート サーバー インスタンスの名前を表す **String** オブジェクト。
