# IsInitialized プロパティ (WMI MSReportServer_ConfigurationSetting)
  レポート サーバー インスタンスが初期化されているかどうかを示します。 読み取り専用です。  
  
## 構文  
  
```vb  
Public Dim IsInitialized As Boolean  
```  
  
```csharp  
public Boolean DatabaseLogonName;  
```  
  
## プロパティ値  
 レポート サーバー インスタンスが初期化されているかどうかを示す**ブール値**オブジェクト。
