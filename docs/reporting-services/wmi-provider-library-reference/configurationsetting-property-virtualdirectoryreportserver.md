# <a name="configurationsetting-property---virtualdirectoryreportserver"></a>VirtualDirectoryReportServer ConfigurationSetting プロパティ
  レポート サーバー アプリケーションの仮想ディレクトリを返します。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## <a name="property-values"></a>プロパティ値  
 仮想ディレクトリを表す **string** 。
