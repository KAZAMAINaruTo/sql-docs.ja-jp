# EditionID プロパティ (WMI MSReportServer_Instance)
  レポート サーバーのエディションを返します。 読み取り専用です。  
  
## 構文  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## プロパティ値  
 レポート サーバーのエディションを表す**ユニット**。
