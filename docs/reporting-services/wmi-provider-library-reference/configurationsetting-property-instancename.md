# <a name="configurationsetting-property---instancename"></a>InstanceName ConfigurationSetting プロパティ
  特定のコンピューターのレポート サーバー インスタンスの名前を指定します。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## <a name="property-value"></a>プロパティ値  
 レポート サーバー インスタンスの名前を表す **String** オブジェクト。
