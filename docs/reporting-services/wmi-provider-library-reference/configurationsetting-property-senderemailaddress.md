# <a name="configurationsetting-property---senderemailaddress"></a>SenderEmailAddress ConfigurationSetting プロパティ
  レポート サーバーからの電子メール送信に使用するアドレスを取得します。 読み取り専用です。  
  
## <a name="syntax"></a>構文  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## <a name="property-values"></a>プロパティ値  
 レポート サーバーで使用される電子メール アドレスを表す読み取り専用の **String** オブジェクト。
