GUI のウィンドウで使用できる、**プロジェクト設定**と**プロジェクト設定の既定の** ダイアログ ボックス。  
  
-   使用して、**プロジェクト設定**ダイアログ ボックスを現在のプロジェクトにユーザー インターフェイス オプションを設定します。 GUI の設定にアクセスする、**ツール**メニューの **プロジェクト設定**、順にクリック**GUI**左側のウィンドウの下部にあります。

## <a name="options"></a>オプション  
**ソースの行の最大数**  
表示されるデータの行の数を設定、**データ**選択したソース テーブルのタブです。  
  
**既定の**: 100  
  
**対象の行の最大数**  
表示されるデータの行の数を設定、**データ**選択されている対象テーブルのタブです。  
  
**既定の**: 100  
  
**変換後のレポートを表示します。**  
スキーマを変換した後は、レポートを表示するには、True を選択します。 結果として得られる変換レポートには、同じレイアウトと評価レポートと情報が含まれています。  
  
**既定の**: False
