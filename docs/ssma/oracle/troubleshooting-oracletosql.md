# <a name="troubleshooting-oracletosql"></a>トラブルシューティング (OracleToSQL)
移行の問題中間にあるコンソール スクリプトまたはコマンドの実行に実行するは珍しいことはできません。  
  
修正可能であるエラーの関連可能性があります。  
  
1.  関数  
  
2.  手順  
  
3.  表示  
  
4.  トリガー  
  
5.  データ型を変更します。  
  
6.  プロジェクト設定を変更します。  
  
問題を解決するには、コンソールで作成されたプロジェクト ファイルを開いて、SSMA ユーザー インターフェイスを使用しています。
