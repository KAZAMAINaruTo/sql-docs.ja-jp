# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>どのような &#39; SSMA for DB2 の (DB2ToSQL)
このトピックでは、DB2 での変更の各リリース SSMA が一覧表示します。  

## <a name="may-2016"></a>2016 年 5 月  
SSMA の DB2 の 2016 年 5 月リリースには、次の変更が含まれています。  

1.  SQL Server 2016 のサポートが追加されました
2.  DB2 のメモリ内と通常のテーブルの SQL Server のメモリ内と hekaton の機能への変換の追加
3.  DB2 アクセス制御の SQL Server ポリシー オブジェクト (行レベルのセキュリティを DB2) への変換の追加
4.  SQL Server のテンポラル テーブルに DB2 システムでバージョン管理されたテーブルの追加の変換
5.  強化された DB2 パーサーおよび競合回避モジュール
6.  .Net 2.0 のインストーラーのチェックを削除
7.  Db2 インストーラーから不要な *.dll を削除
8.  [保存] プロジェクト"固定と SSMA コンソールの「プロジェクトを開く」コマンド
9.  SSMA コンソールの固定"securepassword"コマンド
10. 初期読み込みのためのオブジェクトのカウントを修正
11. グローバル設定のバグを修正
  
## <a name="march-2016"></a>2016 年 3 月  
DB2 SSMA の 2016 年 3 月プレビューのリリースには、次の変更が含まれています。  
  
1.  SQL Server 2016 への移行をサポートします。  
  
## <a name="january-2016"></a>2016 年 1 月  
DB2 SSMA の 2016 年 1 月メンテナンスのリリースには、次の変更が含まれています。  
  
1.  標準的な関数の数のサポートが追加されました  
  
2.  固定 DB2 パーサー エラー  
  
3.  固定の DB2 v9 zOS (RFC 5690920) のサポート  
  
4.  固定の DB2 の変換中に識別子のエラーを解決できません。  
  
5.  SSMA (RFC 5706203) に追加された View ログ メニュー項目  
  
6.  追加された製品利用統計情報  
  
## <a name="november-2014"></a>2014 年 11 月  
SSMA の DB2 の 2014 年 11 月リリースは、その種類の最初の数値です。
