# <a name="properties-collection-ado"></a>プロパティのコレクション (ADO)
すべてが含まれています、[プロパティ](../../../ado/reference/ado-api/property-object-ado.md)オブジェクトの特定のインスタンスのオブジェクト。  
  
## <a name="remarks"></a>解説  
 ADO オブジェクトによってがある、**プロパティ**コレクションから成る**プロパティ**オブジェクト。 各**プロパティ**オブジェクトは ADO オブジェクト プロバイダーに固有の特性に対応します。  
  
> [!NOTE]
>  参照してください、[プロパティ](../../../ado/reference/ado-api/property-object-ado.md)オブジェクトの使用方法の詳細な説明のトピック**プロパティ**オブジェクト。  
  
 **動的プロパティ**の**Recordset**オブジェクト (使用できなくなります) ときにスコープ外に移動して、**レコード セット**が閉じられます。
