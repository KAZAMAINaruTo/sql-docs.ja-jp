型を指定[レコード](../../../ado/reference/ado-api/record-object-ado.md)オブジェクト。  
  
|定数|値|Description|  
|--------------|-----------|-----------------|  
|**adSimpleRecord**|0|示します、*単純*レコード (子ノードを含まない)。|  
|**adCollectionRecord**|1|示します、*コレクション*レコード (子ノードが含まれています)。|  
|**adRecordUnknown**|-1|示しますこの型**レコード**が不明です。|
