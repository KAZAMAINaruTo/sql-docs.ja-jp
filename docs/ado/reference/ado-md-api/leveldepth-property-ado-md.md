# <a name="leveldepth-property-ado-md"></a>LevelDepth プロパティ (ADO MD)
階層のルートの間のレベルの数を示す、[メンバー](../../../ado/reference/ado-md-api/member-object-ado-md.md)です。  
  
## <a name="return-values"></a>戻り値  
 返します、**長い**整数は読み取り専用とします。  
  
## <a name="remarks"></a>解説  
 使用して、 **LevelDepth**までの距離を決定するプロパティ、[メンバー](../../../ado/reference/ado-md-api/member-object-ado-md.md)階層のルート レベルからのオブジェクト。 **LevelDepth**ルートにあるメンバーのレベルは 0 です。 これに対応して、[深さ](../../../ado/reference/ado-md-api/depth-property-ado-md.md)のプロパティ、[レベル](../../../ado/reference/ado-md-api/level-object-ado-md.md)オブジェクト。
