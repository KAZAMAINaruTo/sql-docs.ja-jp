# <a name="formattedvalue-property-ado-md"></a>FormattedValue プロパティ (ADO MD)
表示の書式設定を示す、[セル](../../../ado/reference/ado-md-api/cell-object-ado-md.md)値。  
  
## <a name="return-values"></a>戻り値

 使用して、 **FormattedValue**の表示の書式設定値を取得するプロパティ、[値](../../../ado/reference/ado-md-api/value-property-ado-md.md)のプロパティ、[セル](../../../ado/reference/ado-md-api/cell-object-ado-md.md)オブジェクト。 たとえば、セルの値が 1056.87 し、この値をドル金額を表す**FormattedValue** $1,056.87 になります。
