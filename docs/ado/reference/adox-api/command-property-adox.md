# <a name="command-property-adox"></a>コマンドのプロパティ (ADOX)
ADO を示す[コマンド](../../../ado/reference/ado-api/command-object-ado.md)を作成またはプロシージャを実行するために使用できるオブジェクト。  
  
## <a name="settings-and-return-values"></a>設定と戻り値  
 有効な ADO を取得または設定**コマンド**オブジェクト。  
  
## <a name="remarks"></a>コメント  
 取得して、プロバイダーが永続的なコマンドをサポートしていない場合、このプロパティの設定時にエラーが発生します。
