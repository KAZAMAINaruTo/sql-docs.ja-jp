# <a name="sortorderenum"></a>SortOrderEnum
インデックス付き列の並べ替え順序を指定します。  
  
|定数|値|Description|  
|--------------|-----------|-----------------|  
|**adSortAscending**|@shouldalert|既定値です。 列の並べ替え順序は昇順です。|  
|**adSortDescending**|2|列の並べ替え順序は降順です。|
