# <a name="indexnulls-property-adox"></a>IndexNulls プロパティ (ADOX)
Null 値を持つインデックス フィールドにレコードがインデックス エントリを持つかどうかを示します。  
  
## <a name="settings-and-return-values"></a>設定と戻り値  
 設定を返します、 [AllowNullsEnum](../../../ado/reference/adox-api/allownullsenum.md)値。 既定値は**adIndexNullsDisallow**です。  
  
## <a name="remarks"></a>解説  
 このプロパティは読み取り専用で[インデックス](../../../ado/reference/adox-api/index-object-adox.md)オブジェクトをコレクションに既に追加されています。
