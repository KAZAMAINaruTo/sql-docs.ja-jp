# <a name="keytypeenum"></a>KeyTypeEnum
型を指定[キー](../../../ado/reference/adox-api/key-object-adox.md): プライマリ、外部キー、または一意です。  
  
|定数|値|Description|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|既定値です。 キーは、主キーです。|  
|**adKeyForeign**|2|キーは、外部キーです。|  
|**adKeyUnique**|3|キーは一意です。|
