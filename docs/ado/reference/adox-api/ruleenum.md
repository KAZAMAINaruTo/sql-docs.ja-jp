# <a name="ruleenum"></a>RuleEnum
ときに従うルールを指定します、[キー](../../../ado/reference/adox-api/key-object-adox.md)を削除します。  
  
|定数|値|Description|  
|--------------|-----------|-----------------|  
|**adRICascade**|1|Cascade 変更します。|  
|**adRINone**|0|既定値です。 アクションが行われません。|  
|**adRISetDefault**|3|外部キーの値は、既定値に設定されます。|  
|**adRISetNull**|2|外部キーの値の設定を null にします。|
