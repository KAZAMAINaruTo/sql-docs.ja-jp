# データの保存、移動および変換に関する Microsoft テクノロジの概要
**概要:** この記事の目的は、データの保存、移動、および変換に関する Microsoft テクノロジの概要を示すことです。 これらのテクノロジではビジネス インテリジェンスがサポートされます。これにより、組織ではクラウドやオンプレミスでの分析およびレポート用の品質データ、および異種データを保存して、アクセスすることができます。  
  
Microsoft ビジネス インテリジェンス テクノロジの概要については、「[Microsoft の BI レポートおよび分析用ツールの概要](https://msdn.microsoft.com/en-us/library/dn655131.aspx)」を参照してください。
