# <a name="dmx-statements---data-definition"></a>DMX ステートメントのデータの定義 
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  次の表は、データ マイニング拡張機能 (DMX) のデータ マイニング データ定義言語 (DDL) の一部であるステートメントについて示しています。  
  
|ステートメントから削除してください。|Description|  
|---------------|-----------------|  
|[マイニング構造 (&) #40";"DMX"&"#41; を作成します。](../dmx/create-mining-structure-dmx.md)|データベースに新しいマイニング構造を作成します。|  
|[マイニング構造の変更 (&) #40";"DMX"&"#41;](../dmx/alter-mining-structure-dmx.md)|既存のマイニング構造にマイニング モデルを追加します。|  
|[マイニング モデル (&) #40";"DMX"&"#41; を作成します。](../dmx/create-mining-model-dmx.md)|データベースに新しいマイニング構造とマイニング モデルを作成します。|  
|[マイニング モデルの削除 (&) #40";"DMX"&"#41;](../dmx/drop-mining-model-dmx.md)|データベースからマイニング モデルを削除します。|  
|[マイニング構造の削除 (&) #40";"DMX"&"#41;](../dmx/drop-mining-structure-dmx.md)|データベースからマイニング構造を削除します。|  
|[エクスポート (&) #40";"DMX"&"#41;](../dmx/export-dmx.md)|マイニング モデルまたはマイニング構造、およびそれと関連付けられたオブジェクトを .abf ファイルにエクスポートします。|  
|[インポート (&) #40";"DMX"&"#41;](../dmx/import-dmx.md)|マイニング モデルまたはマイニング構造、およびそれと関連付けられたオブジェクトを .abf ファイルからインポートします。|  
|[SELECT INTO & #40";"DMX"&"#41;](../dmx/select-into-dmx.md)|既存のマイニング モデルのコピーを作成します。|
