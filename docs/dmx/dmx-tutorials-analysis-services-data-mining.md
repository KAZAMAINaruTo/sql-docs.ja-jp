# <a name="dmx-tutorials-analysis-services---data-mining"></a>DMX のチュートリアル (Analysis Services - データ マイニング)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  以下のチュートリアルでは、データ マイニング構造およびデータ マイニング モデルで、データ マイニング拡張機能 (DMX) ステートメントを使用する方法について学習します。  
  
## <a name="in-this-section"></a>このセクションの内容  
 [Bike Buyer DMX のチュートリアル](http://msdn.microsoft.com/library/4b634cc1-86dc-42ec-9804-a19292fe8448)  
  
 このチュートリアルでは、DMX クエリ言語を使用して、マイニング モデルの作成、トレーニング、および調査を行う方法を学習します。 その後、これらのマイニング モデルを使用して、顧客が特定の製品を購入する可能性があるかどうかに関する予測を作成します。  
  
 [マーケット バスケット DMX のチュートリアル](http://msdn.microsoft.com/library/6e262a1d-c89e-4033-8368-46cf25168ef5)  
  
 このチュートリアルでは、同時に購入される傾向が高い製品を予測するマイニング モデルを作成する方法を学習します。 また、このチュートリアルでは、データ マイニングで入れ子になったテーブルの使用方法についても示します。
