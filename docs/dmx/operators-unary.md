# <a name="operators---unary"></a>単項の演算子
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  単項演算子は、1 つのオペランドに基づいた演算を実行します。たとえば、数値式の負または正の値が返されます。  
  
 次の表は、DMX がサポートする単項演算子について示しています。  
  
|演算子|Description|  
|--------------|-----------------|  
|[-(& a) #40 です。負の値&#41;&#40;です。MDX と #41 です。](../mdx/negative-mdx.md)|数値式の負の値を返します。|  
|[+ (& a) #40 です。正の値&#41;&#40;です。MDX と #41 です。](../mdx/positive-mdx.md)|数値式の正の値を返します。|
