# <a name="operators---logical"></a>演算子の論理
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  値を評価しでのブール値を返す、データ マイニング拡張機能 (DMX) 式で論理演算子を使用できる[!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]です。  
  
 次の表は、DMX がサポートする論理演算子について示しています。  
  
|演算子|Description|  
|--------------|-----------------|  
|[および & # #40; DMX &#41;](../dmx/and-dmx.md)|2 つの数値式の論理積演算を実行します。|  
|[いない & # #40; DMX &#41;](../dmx/not-dmx.md)|1 つの数値式の論理否定を実行します。|  
|[または & # #40; DMX &#41;](../dmx/or-dmx.md)|2 つの数値式の論理和を実行します。|
