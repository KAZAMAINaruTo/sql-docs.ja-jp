# <a name="mdx-data-definition---drop-subcube"></a>MDX データ定義、DROP SUBCUBE
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  指定されたサブキューブを削除し、指定された名前の以前のキューブ定義またはサブキューブ定義に戻します。  
  
## <a name="syntax"></a>構文  
  
```  
  
DROP SUBCUBE Subcube_Name   
```  
  
## <a name="arguments"></a>引数  
 *Subcube_Name*  
 サブキューブ名を指定する有効な文字列式です。
