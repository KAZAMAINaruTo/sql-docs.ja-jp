# <a name="mdx-scripting-statements-mdx"></a>MDX スクリプト ステートメント (MDX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  多次元式 (MDX) では、以下のステートメントによって、MDX スクリプト内のコンテキスト、スコープ、フロー制御を管理します。  
  
## <a name="in-this-section"></a>このセクションの内容  
  
|トピック|説明|  
|-----------|-----------------|  
|[CALCULATE ステートメント&#40;MDX&#41;](../mdx/mdx-scripting-calculate.md)|サブキューブを計算します。サブキューブ内に組み込まれているディメンションの解決順序を指定することも可能です。|  
|[CASE ステートメント&#40;MDX&#41;](../mdx/case-statement-mdx.md)|複数の比較から、条件付きで特定の値を返します。|  
|[EXISTING キーワード&#40;MDX&#41;](../analysis-services/multidimensional-models/mdx/mdx-query-existing-keyword.md)|指定されたセットを現在のコンテキストで評価するように設定します。|  
|[FREEZE ステートメント&#40;MDX&#41;](../mdx/mdx-scripting-freeze.md)|指定されたサブキューブのセル値を現在の値にロックします。|  
|[IF ステートメント&#40;MDX&#41;](../mdx/mdx-scripting-if.md)|条件が満たされる場合、ステートメントを実行します。|  
|[SCOPE ステートメント&#40;MDX&#41;](../mdx/mdx-scripting-scope.md)|指定された MDX ステートメントのスコープを指定されたサブキューブに限定します。|
