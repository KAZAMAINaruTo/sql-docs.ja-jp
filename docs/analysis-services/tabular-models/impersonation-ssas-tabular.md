データをインポートまたは処理されるときに、権限借用の資格情報は、データ ソースへの接続し、データのフェッチに使用されます。 これは、*サーバー側*ワークスペース データベースをホストしている Analysis Services サーバーがデータ ソースに接続し、データをフェッチするために、クライアント アプリケーションのコンテキストで実行中の操作です。
