# <a name="programming-administrative-tasks-with-amo"></a>AMO による管理タスクのプログラミング
[!INCLUDE[ssas-appliesto-sqlas](../../../includes/ssas-appliesto-sqlas.md)]分析管理オブジェクト (AMO) が管理するクライアント アプリケーションから使用されるプログラミング ライブラリ[!INCLUDE[ssASnoversion](../../../includes/ssasnoversion-md.md)]です。 ここでは、AMO オブジェクトを使用したプログラミング方法について説明します。  
  
 AMO ライブラリは、特定のタスクの実行に使用される、論理的に関連したオブジェクトのグループと考えることができます。
