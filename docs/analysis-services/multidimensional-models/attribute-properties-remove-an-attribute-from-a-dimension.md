# <a name="attribute-properties---remove-an-attribute-from-a-dimension"></a>属性のプロパティ - ディメンションから属性を削除
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]内のディメンションから属性を削除する[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)]、内の属性を右クリックし、**属性**のペイン、**ディメンション構造**ディメンションのタブデザイナーで[!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]、クリックして**削除**です。 **[オブジェクトの削除]** ダイアログ ボックスで **[OK]**をクリックします。 これでディメンションから属性が削除されますが、ディメンションのデータ ソース ビューには影響を与えません。
