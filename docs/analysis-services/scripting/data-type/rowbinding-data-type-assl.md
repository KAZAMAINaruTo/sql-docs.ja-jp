# <a name="rowbinding-data-type-assl"></a>RowBinding データ型 (ASSL)

## <a name="syntax"></a>構文  
  
```xml

   <!-- The following elements extend Binding -->  
   <TableID>...</TableID>  
</RowBinding>  
```
