# <a name="goal-element-assl"></a>Goal 要素 (ASSL)
  目標を識別、 [Kpi](../../../analysis-services/scripting/objects/kpi-element-assl.md)要素。  
  
## <a name="syntax"></a>構文  
  
```xml  
  
<Kpi>  
   ...  
   <Goal>...</Goal>  
   ...  
</Kpi>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|文字列|  
|既定値|なし|  
|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[Kpi](../../../analysis-services/scripting/objects/kpi-element-assl.md)|  
|子要素|なし|  
  
## <a name="remarks"></a>解説  
 **目標**要素には、多次元式 (MDX) 式が含まれています。  
  
 親に対応する要素**目標**分析管理オブジェクト (AMO) オブジェクト モデルは<xref:Microsoft.AnalysisServices.Kpi>します。
