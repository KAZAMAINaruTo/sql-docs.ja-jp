# <a name="datasize-element-assl"></a>DataSize 要素 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<DataItem>  
   ...  
   <DataSize>...</DataSize>  
   ...  
</DataItem  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|Integer|

|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|
