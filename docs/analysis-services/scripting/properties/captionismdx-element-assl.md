# <a name="captionismdx-element-assl"></a>CaptionIsMdx 要素 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<Action>  
   ...  
   <CaptionIsMdx>...</CaptionIsMdx>  
   ...  
</Action>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|
