# <a name="keyuniquenessguarantee-element-assl"></a>KeyUniquenessGuarantee 要素 (ASSL)

```xml  
  
<DimensionAttribute>

   <KeyUniquenessGuarantee>...</KeyUniquenessGuarantee>
```  
