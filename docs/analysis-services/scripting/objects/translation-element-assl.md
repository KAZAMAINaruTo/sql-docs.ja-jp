# <a name="translation-element-assl"></a>Translation 要素 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<Translations>  
   <Translation xsi:type="Translation">...</Translation>  
   <!-- or -->  
   <Translation xsi:type="AttributeTranslation">...</Translation>  
</Translations>  
```

|特性|説明|
