# <a name="miningmodelpermissions-element-assl"></a>MiningModelPermissions 要素 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<MiningModel>  
   ...  
   <MiningModelPermissions>  
      <MiningModelPermission xsi:type="Permission">...</MiningModelPermission>  
   </MiningModelPermissions>  
   ...  
</MiningModel>  
```
