# Analysis Services の機能とタスク
  Analysis Services に関する基本ドキュメントは、インストールしたモードで使用できるツール、タスク、および機能に関する情報だけを参照できるように、モデルの種類およびサーバーのモード別に分類されています。 複数のモードに関連するサーバー管理タスクは、インスタンス管理にあります。
