# <a name="lastschemaupdate-element-xmla"></a>LastSchemaUpdate 要素 (XMLA)

## <a name="syntax"></a>構文  
  
```xml  
  
<Cube>  
   ...  
   <LastSchemaUpdate>...</LastSchemaUpdate>  
   ...  
</Cube>  
```
