# <a name="description-element-xmla"></a>Description 要素 (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]
  親の説明を表す[エラー](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)要素。  
  
## <a name="syntax"></a>構文  
  
```xml  
  
<Error>  
   ...  
   <Description>...</Description>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|文字列|  
|既定値|なし|  
|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[[エラー]](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)|  
|子要素|なし|
