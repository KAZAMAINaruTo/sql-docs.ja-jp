# <a name="name-element-parameter-xmla"></a>Name 要素 (Parameter) (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]によって表されるパラメーターの名前を含む、[パラメーター](../../../analysis-services/xmla/xml-elements-properties/parameter-element-xmla.md)要素。  
  
## <a name="syntax"></a>構文  
  
```  
  
<Parameter>  
   ...  
   <Name>...</Name>  
   ...  
</Parameter>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|Description|  
|--------------------|-----------------|  
|データ型と長さ|String|  
|既定値|なし|  
|基数|1-1 : 必須要素で、1 回だけ出現します|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[パラメーター](../../../analysis-services/xmla/xml-elements-properties/parameter-element-xmla.md)|  
|子要素|なし|
