# <a name="errors-and-warnings-event-category"></a>エラーと警告イベント カテゴリ

  エラーと警告イベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|Description|  
|-----------------|--------------|-----------------|  
|[エラー]|17|トレースが開始されてからのすべての新しい Error イベントを記録します。|  
  
 エラーと警告の各イベント クラスに関連する列については、「 [Errors and Warnings イベントのデータ列](../../analysis-services/trace-events/errors-and-warnings-events-data-columns.md)」を参照してください。
