# <a name="session-events-event-category"></a>Session イベントのイベント カテゴリ
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]Session イベントのイベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|Description|  
|-----------------|--------------|-----------------|  
|Existing Connection|41|トレースが開始されてからのすべての接続イベントを収集します。|  
|Existing Session|42|トレースが開始されてからのすべての Session イベントを収集します。|  
|Session Initialize|43|セッション初期化イベントを収集します。|  
  
 Session イベントの各イベント クラスに関連する列の詳細については、「 [Session イベントのデータ列](../../analysis-services/trace-events/session-events-data-columns.md)」を参照してください。
