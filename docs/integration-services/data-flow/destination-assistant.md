# デスティネーション アシスタント
  変換先アシスタント コンポーネントを使用すると、変換先コンポーネントおよび接続マネージャーを作成できます。 このコンポーネントは、SSIS ツールボックスの **[お気に入り]** セクションにあります。  
  
> [!NOTE]  
>  変換先アシスタントは、Integration Services 接続プロジェクトと対応するウィザードに置き換わるものです。  
  
## 関連タスク  
 変換先アシスタントを使用して変換先および接続マネージャーを追加する方法については、「 [Add a Destination using Destination Assistant](../Topic/Add%20a%20Destination%20using%20Destination%20Assistant.md)」を参照してください。
