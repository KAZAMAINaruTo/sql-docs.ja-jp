# <a name="certification-by-sap"></a>SAP による認定
  Microsoft Connector 1.1 for SAP BW は、SAP から SAP NetWeaver との統合に関する認定を受けました。  
  
 ![証明される SAP - SAP NetWeaver での統合](../integration-services/media/sapcertifiedforssis11.gif "証明される SAP - SAP NetWeaver での統合")  
  
 次の表に、認定の詳細について説明します。  
  
|SAP インターフェイス|SAP リリース レベル|認定日|関連するコンポーネント|  
|-------------------|------------------------|------------------------|-----------------------|  
|BW-OHS 7.0 - SAP NetWeaver Business Intelligence - オープン ハブ サービス 7.0|Business Intelligence 7.0|2012 年 12 月|ソース|  
|BW-STA 3.5 - SAP BW 3.5 用ステージング BAPI|Business Intelligence 3.5、7.0|2012 年 12 月|転送先|
