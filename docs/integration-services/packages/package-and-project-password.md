# パッケージおよびプロジェクトのパスワード
  **[パッケージ パスワード]** ダイアログ ボックスを使用すると、パスワードによって暗号化されるパッケージのパッケージ パスワードを指定できます。 パッケージで使用される保護レベルが **[機微なデータをパスワードで暗号化する]**または **[すべてのデータをパスワードで暗号化する]** の場合、パスワードを指定する必要があります。  
  
## オプション  
 **Password**  
 パスワードを入力します。
