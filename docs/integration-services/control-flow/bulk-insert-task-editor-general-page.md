# [一括挿入タスク エディター] ([全般] ページ)
  **[一括挿入タスク エディター]** ダイアログ ボックスの **[全般]** ページを使用して、一括挿入タスクの名前と説明を指定します。  
  
 このタスクの詳細については、「 [Bulk Insert Task](../../integration-services/control-flow/bulk-insert-task.md)」を参照してください。  
  
## オプション  
 **名前**  
 一括挿入タスクに一意の名前を指定します。 この名前は、タスク アイコンのラベルとして使用されます。  
  
> [!NOTE]  
>  タスク名はパッケージ内で一意である必要があります。  
  
 **Description**  
 一括挿入タスクの説明を入力します。
