# <a name="azure-hdinsight-connection-manager"></a>Azure HDInsight 接続マネージャー
**Azure HDInsight 接続マネージャー** SSIS パッケージを Azure の HDInsight クラスターへの接続を有効にします。

**Azure HDInsight 接続マネージャー**のコンポーネントである、 [SQL Server Integration Services (SSIS) Feature Pack for Azure](../../integration-services/azure-feature-pack-for-integration-services-ssis.md)です。

作成および構成、 **Azure HDInsight 接続マネージャー**、以下の手順に従います。

1. **SSIS 接続マネージャーの追加**ダイアログ ボックスで、 **AzureHDInsight**、 をクリック**追加**です。
2. **Azure HDInsight 接続マネージャー エディター**  ダイアログ ボックスで、指定、**クラスター DNS 名**(、プロトコル プレフィックスなし)、 **Username**、および**パスワード**HDInsight クラスターへの接続にします。
3. **[OK]** をクリックして、ダイアログ ボックスを閉じます。
4. 作成した接続マネージャーのプロパティは、 **[プロパティ]** ウィンドウに表示されます。
