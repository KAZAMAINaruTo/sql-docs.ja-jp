# <a name="complete---upgrade"></a>完了 - アップグレード
  [完了] (アップグレード) ページを使用して、アップグレードが完了したことを確認します。  
  
## <a name="options"></a>および  
 このダイアログ ボックスには、この操作のセットアップ ログ ファイルへのリンクが含まれています。 続行するには、 **[閉じる]** をクリックします。
