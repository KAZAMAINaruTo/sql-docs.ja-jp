# <a name="completing-the-wizard-cube-wizard"></a>[ウィザードの完了] (キューブ ウィザード)
  **[ウィザードの完了]** ページを使用すると、キューブに名前を付け、ウィザードの完了時に作成されるキューブをプレビューできます。  
  
## <a name="options"></a>および  
 **キューブ名**  
 キューブの名前を入力します。  
  
 **プレビュー**  
 ウィザードの完了時に作成されるキューブの構造を表示します。
