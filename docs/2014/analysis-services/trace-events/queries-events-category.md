# <a name="queries-events-category"></a>Queries イベント カテゴリ
  Queries イベントのイベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|説明|  
|-----------------|--------------|-----------------|  
|Query Begin|9|トレースが開始されてからのすべての Query Begin イベントを収集します。|  
|クエリの終了|10|トレースが開始されてからのすべての Query End イベントを収集します。|  
  
 Query イベントの各イベント クラスに関連する列の詳細については、「 [Queries イベントのデータ列](queries-events-data-columns.md)」を参照してください。
