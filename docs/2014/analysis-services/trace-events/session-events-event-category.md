# <a name="session-events-event-category"></a>Session イベントのイベント カテゴリ
  Session イベントのイベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|説明|  
|-----------------|--------------|-----------------|  
|Existing Connection|41|トレースが開始されてからのすべての接続イベントを収集します。|  
|Existing Session|42|トレースが開始されてからのすべての Session イベントを収集します。|  
|Session Initialize|43|セッション初期化イベントを収集します。|  
  
 Session イベントの各イベント クラスに関連する列の詳細については、「 [Session イベントのデータ列](session-events-data-columns.md)」を参照してください。
