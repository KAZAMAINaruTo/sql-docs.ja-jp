# <a name="file-load-and-save-event-category"></a>ファイルの読み込みと保存のイベント カテゴリ
  ファイルの読み込みと保存のイベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|説明|  
|-----------------|--------------|-----------------|  
|ファイルの読み込みの開始|90|ファイルの読み込みの開始|  
|ファイルの読み込みの終了|91|ファイルの読み込みの終了|  
|ファイルの保存の開始|92|ファイルの保存の開始|  
|ファイルの保存の終了|93|ファイルの保存の終了|  
|ページアウトの開始|94|ページアウトの開始|  
|ページアウトの終了|95|ページアウトの終了|  
|ページインの開始|96|ページインの開始|  
|ページインの終了|97|ページインの終了|  
  
 ファイルの読み込みと保存の各イベント クラスに関連する列については、「 [File Load and Save Data Columns](file-load-and-save-data-columns.md)」を参照してください。
