# <a name="notification-events-event-category"></a>Notification イベントのイベント カテゴリ
  Notification イベントのイベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|説明|  
|-----------------|--------------|-----------------|  
|Notification|39|トレースが開始されてからのすべての Notification イベントを収集します。|  
|User Defined|40|トレースが開始されてからのすべてのユーザー定義イベントを収集します。|  
  
 Notification イベントの各イベント クラスに関連する列については、「 [Notification イベントのデータ列](notification-events-data-columns.md)」を参照してください。
