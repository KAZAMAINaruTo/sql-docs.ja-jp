# <a name="command-events-event-category"></a>Command イベントのイベント カテゴリ
  Command イベントのイベント カテゴリには、次の表に示したイベント クラスがあります。  
  
|Event Class|イベント ID|説明|  
|-----------------|--------------|-----------------|  
|Command Begin|15|トレースが開始されてからのすべての Command Begin イベントを収集します。|  
|Command End|16|トレースが開始されてからのすべての Command End イベントを収集します。|  
  
 Command イベントの各イベント クラスに関連する列については、「 [Command イベントのデータ列](command-events-data-columns.md)」を参照してください。
