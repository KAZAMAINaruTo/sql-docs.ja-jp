# <a name="miningstructurecolumn-data-type-assl"></a>MiningStructureColumn データ型 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<MiningStructureColumn>  
   <Name>...</Name>  
   <ID>...</ID>  
   <Description>...</Description>  
   <Type>...</Type>  
   <Annotations>...</Annotations>  
</MiningStructureColumn>  
```
