# <a name="querybinding-data-type-assl"></a>QueryBinding データ型 (ASSL)

## <a name="syntax"></a>構文  
  
```xml  
  
<QueryBinding>  
   <!-- The following elements extend TabularBinding -->  
   <DataSourceID>...</DataSourceID>  
      <QueryDefinition>...</QueryDefinition>  
</QueryBinding>  
```
