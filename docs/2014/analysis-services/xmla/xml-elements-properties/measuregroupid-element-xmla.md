# <a name="measuregroupid-element-xmla"></a>MeasureGroupID 要素 (XMLA)
  オブジェクト参照を含む親要素内で、メジャー グループを識別します。  
  
## <a name="syntax"></a>構文  
  
```xml  
  
<Object> <!-- or one of the elements listed below in the Element Relationships table -->  
   ...  
   <MeasureGroupID>...</MeasureGroupID>  
   ...  
</Object>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|String|  
|既定値|なし|  
|Cardinality|[Source](source-element-xmla.md)、[Target](../xml-elements-properties/target-element-xmla.md) = 1-1: 必須要素で、1 回だけ出現します。|  
|Cardinality|他のすべて = 0-1: 省略可能な要素で、出現する場合は 1 回だけの出現が可能です。|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[Object](object-element-xmla.md)、[ParentObject](parentobject-element-xmla.md)、[Source](source-element-xmla.md)、[Target](../xml-elements-properties/target-element-xmla.md)|  
|子要素|なし|
