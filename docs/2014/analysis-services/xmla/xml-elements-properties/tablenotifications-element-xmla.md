# <a name="tablenotifications-element-xmla"></a>TableNotifications 要素 (XMLA)
  コレクションを格納[TableNotification](tablenotification-element-xmla.md)によって使用される要素、 [NotifyTableChange](../xml-elements-commands/notifytablechange-element-xmla.md)コマンド。  
  
## <a name="syntax"></a>構文  
  
```xml  
  
<NotifyTableChange >  
   ...  
   <TableNotifications>  
      <TableNotification>...</TableNotification>  
   </TableNotifications>  
   ...  
</NotifyTableChange>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|なし|  
|既定値|なし|  
|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[NotifyTableChange](../xml-elements-commands/notifytablechange-element-xmla.md)|  
|子要素|[TableNotification](tablenotification-element-xmla.md)|
