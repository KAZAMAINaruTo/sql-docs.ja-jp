# <a name="applycompression-element-xmla"></a>ApplyCompression 要素 (XMLA)
  指定するかどうか、親[バックアップ](../xml-elements-commands/backup-element-xmla.md)コマンドがバックアップ ファイルを圧縮します。  
  
## <a name="syntax"></a>構文  
  
```xml  
  
<Backup>  
   ...  
   <ApplyCompression>...</ApplyCompression>  
   ...  
</Backup>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|ブール値|  
|既定値|True|  
|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[バックアップ](../xml-elements-commands/backup-element-xmla.md)|  
|子要素|なし|
