# <a name="time-element-xmla"></a>Time 要素 (XMLA)
  によって使用される制限時間を指定します、 [DesignAggregations](../xml-elements-commands/designaggregations-element-xmla.md)集計をデザインするコマンド。  
  
## <a name="syntax"></a>構文  
  
```xml  
  
<DesignAggregations>  
   ...  
   <Time>...</Time>  
   ...  
</DesignAggregations>  
```  
  
## <a name="element-characteristics"></a>要素の特性  
  
|特性|説明|  
|--------------------|-----------------|  
|データ型と長さ|Duration|  
|既定値|なし|  
|Cardinality|0-1 : 省略可能な要素で、出現する場合は 1 回だけの出現が可能です|  
  
## <a name="element-relationships"></a>要素の関係  
  
|リレーションシップ|要素|  
|------------------|-------------|  
|親要素|[DesignAggregations](../xml-elements-commands/designaggregations-element-xmla.md)|  
|子要素|なし|
