# <a name="backup-checksum-default"></a>バックアップ チェックサムの既定
  バックアップと復元中にバックアップ チェックサムを有効または無効にするために、バックアップ チェックサムの既定の設定を使用します。  
  
 次の表に、このオプションで使用可能な値を示します。  
  
|||  
|-|-|  
|**[値]**|**意味**|  
|0|無効。 これが既定の設定です。|  
|1|有効|  
  
 この設定はすぐに有効になります。
