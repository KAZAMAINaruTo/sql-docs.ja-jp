# <a name="master-data-services"></a>マスター データ サービス
  [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)] (MDS) は、マスター データ管理のための [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] ソリューションです。 マスター データ管理 (MDM) とは、メンテナンス可能な形でマスターのリストをまとめることを目標に、データの非トランザクション リストを探索および定義するために組織が必要とする作業を表します。 MDM プロジェクトには、一般に、MDM テクノロジの実装と共に内部ビジネス プロセスの評価および再構築が含まれます。 優れた MDM ソリューションをビジネスの現場に導入することで、信頼性が高く分析可能なデータを一元的に管理できるようになるため、結果として、より的確な意思決定を行うことが可能になります。  
  
 ほとんどのビジネス ユーザーは適切なトレーニングを受けることで [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)] ソリューションを実装できるようになります。 また、MDS を使用して、すべてのドメインの管理が可能になります。つまり、顧客、製品、またはアカウントのリスト管理のみに限定されるソリューションではありません。 MDS の初回インストール時には、ドメインの構造は含まれません。必要なドメインの定義は、ドメインのモデルを作成することによって行います。  
  
 マスター データ サービスのその他の機能には、階層、きめ細かいセキュリティ、トランザクション、データのバージョン管理、ビジネス ルールなどがあります。  
  
 [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)] 次のコンポーネントとツールが含まれます。  
  
-   [!INCLUDE[ssMDScfgmgr](../includes/ssmdscfgmgr-md.md)]を作成および構成に使用するツール[!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)]データベースと web アプリケーションです。  
  
-   [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)]を使用して管理タスク (モデルやビジネス ルールの作成)、などを実行して、データを更新するユーザーにアクセスする web アプリケーションです。  
  
-   MDSModelDeploy.exe: モデル オブジェクトとデータのパッケージを作成し、他の環境に配置できるようにするツールです。  
  
-   [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)] web サービスを拡張または用のカスタム ソリューションを開発する開発者が使用できる[!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)]です。  
  
-   [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)] [!INCLUDE[ssMDSXLS](../includes/ssmdsxls-md.md)]: これを使用して、データの管理や新しいエンティティおよび属性の作成を行うことができます。  
  
 MDS のリソースの概要は、次を参照してください。、 [SQL Server マスター データ サービスのポータル](http://go.microsoft.com/fwlink/?LinkID=214272)です。
