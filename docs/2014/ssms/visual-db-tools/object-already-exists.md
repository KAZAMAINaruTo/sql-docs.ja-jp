# <a name="object-already-exists"></a>[オブジェクトが既に存在します]
  保存しようとしているオブジェクトと同じ名前のオブジェクトが既にデータベースに存在します。 別の名前を選択し直してください。  
  
 列名が既に存在することを示す場合もあります。
