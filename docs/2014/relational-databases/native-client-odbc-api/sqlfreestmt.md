# <a name="sqlfreestmt"></a>SQLFreeStmt
  **SQLFreeStmt** ODBC 3.0 以降ではお勧めしません。 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client ODBC ドライバーですべて定義されている*オプション*値**SQLFreeStmt**です。 ただし、 [SQLCloseCursor](sqlclosecursor.md)、 [SQLBindParameter](sqlbindparameter.md)、 [SQLBindCol](sqlbindcol.md)、 **SQLSetDescField**、および[SQLFreeHandle](sqlfreehandle.md)複製の機能を置き換えたり、 **SQLFreeStmt**代わりに使用する必要があります。
