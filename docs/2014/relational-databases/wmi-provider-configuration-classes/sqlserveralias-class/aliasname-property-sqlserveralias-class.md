## <a name="syntax"></a>構文  
  
```  
  
object  
.AliasName [= value]  
```  
  
## <a name="parts"></a>要素  
 *object*  
 [の別名を表す](sqlserveralias-class.md) SqlServerAlias クラス [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] オブジェクト。
