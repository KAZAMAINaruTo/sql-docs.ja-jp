# <a name="ipaddressname-property-servernetworkprotocolipaddress-class"></a>IpAddressName プロパティ (ServerNetworkProtocolIPAddress クラス)
  IP アドレスの名前を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
  
object  
.IpAddressName [= value]  
```  
  
## <a name="parts"></a>要素  
 *object*  
 A [ServerNetworkProtocolIPAdress クラス](servernetworkprotocolipaddress-class.md)のインスタンス上のネットワーク プロトコルの IP アドレスを表すオブジェクトを[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]します。  
  
## <a name="property-valuereturn-value"></a>プロパティ値/戻り値  
 IP アドレスの名前を指定する文字列値。
