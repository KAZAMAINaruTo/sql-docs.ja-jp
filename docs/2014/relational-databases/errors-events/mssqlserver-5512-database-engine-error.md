# <a name="mssqlserver5512"></a>MSSQLSERVER_5512
    
## <a name="details"></a>詳細  
  
|||  
|-|-|  
|製品名|MSSQLSERVER|  
|イベント ID|5512|  
|イベント ソース|MSSQLSERVER|  
|コンポーネント|SQLEngine|  
|シンボル名|FS_TRUNCATEDIR_FAILED|  
|メッセージ テキスト|ディレクトリ ''%.*ls'' の切り捨て中にエラー 0x%x (%ls) が発生しました。|
