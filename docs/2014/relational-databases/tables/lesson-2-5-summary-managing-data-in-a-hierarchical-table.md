# <a name="summary-managing-data-in-a-hierarchical-table"></a>まとめ : 階層テーブルでのデータの管理
  `hierarchyid` データ型を使用すると、階層データを反映するテーブルを簡単に作成できるようになり、深さ優先および幅優先のインデックスを作成してテーブルに効率的にアクセスできます。 データをクエリおよび操作する場合は、階層的な手法を使用します。
