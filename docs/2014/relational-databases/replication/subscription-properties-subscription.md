# <a name="subscription-properties---ltsubscriptiongt"></a>サブスクリプションのプロパティ - &lt;Subscription&gt;
  ここでは **[サブスクリプションのプロパティ]** ダイアログ ボックスに関する次の情報について紹介します。  
  
-   「[サブスクリプションのプロパティ - サブスクライバー](subscription-properties-subscriber.md)」ではサブスクライバーで使用できるサブスクリプションのプロパティについて説明します (プル サブスクリプションのみ)。  
  
-   「[Subscription Properties - Publisher](subscription-properties-publisher.md) 」ではパブリッシャーで使用できるサブスクリプションのプロパティについて説明します (プル サブスクリプションおよびプッシュ サブスクリプションのみ)。
