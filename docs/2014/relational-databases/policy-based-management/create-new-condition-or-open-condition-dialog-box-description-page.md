# <a name="create-new-condition-or-open-condition-dialog-box-description-page"></a>[新しい条件の作成] または [条件を開く] ダイアログ ボックスの [説明] ページ
  このダイアログ ボックスを使用すると、ポリシー ベースの管理条件に説明を追加できます。  
  
## <a name="options"></a>および  
 **[説明]**  
 条件の説明を入力します。  
  
 **作成日**  
 条件が作成された日付。  
  
 **作成者**  
 条件を作成したログイン。  
  
 **[更新日]**  
 条件が最後に変更された日付。  
  
 **[変更者]**  
 条件を最後に変更したログイン。
