# <a name="task-3-verifying-the-data-in-master-data-manager"></a>タスク 3: マスター データ マネージャーのデータを確認する
  このタスクのことを確認する、**業者**エンティティが作成**MDS**を使用して**マスター データ マネージャー Web アプリケーション**です。  
  
1.  場合**マスター データ マネージャー**が既に開かれているをクリックして**SQL Server 2012 マスター データ サービス**上部のホーム ページに移動します。 それ以外の場合に移動[ http://localhost/mds ](http://localhost/mds)を起動する**マスター データ マネージャー**です。  
  
2.  選択**Suppliers**の**モデル**、 をクリック**エクスプ ローラー**です。  
  
     ![マスター データ マネージャー、エクスプ ローラー ボタン](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "マスター データ マネージャー、エクスプ ローラー ボタン")  
  
3.  MDS に格納されているデータを確認します。 データが表示されない場合は、選択されていることを確認**Suppliers**の**モデル**起動する前に、ホーム ページに**エクスプ ローラー**です。 追加するかを使用して仕入先の一覧から削除**メンバーの追加**と**メンバーの削除**ツールバーのボタンです。
