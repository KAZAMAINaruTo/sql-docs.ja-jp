# <a name="report-designer-f1-help"></a>レポート デザイナーの F1 ヘルプ
  このセクションでは、F1 ヘルプについて、 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]レポート デザイナーのウィザード、ビュー、およびダイアログ ボックス。 ダイアログ ボックスの使用中は、F1 キーを押すか、ダイアログ ボックスに表示される [ヘルプ] をクリックすることで、F1 ヘルプを開くことができます。
