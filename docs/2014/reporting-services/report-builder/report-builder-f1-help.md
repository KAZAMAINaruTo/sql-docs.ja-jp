# <a name="report-builder-f1-help"></a>レポート ビルダーの F1 ヘルプ
  このセクションには、レポート ビルダーのウィザード、ビュー、ペイン、およびダイアログ ボックスのヘルプが含まれています。 F1 キーを押すか ダイアログ ボックスで ヘルプ ボタンをクリックしてヘルプを開くこともできます。
