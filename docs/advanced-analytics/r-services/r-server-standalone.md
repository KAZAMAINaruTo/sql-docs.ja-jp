# R Server (Standalone)
  **Microsoft R サーバー (スタンドアロン)** Microsoft R サービスによって提供される企業向け分析プラットフォームの一部です。  
  
 R は、サポート、スケーラビリティ、およびセキュリティを提供することによって拡張します。 [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] 並列を追加してユーザーをエンタープライズ規模でデータを分析し、データがメモリに収まらない非常に大きくなるを使用して、データのチャンクの処理もオープン ソース R のメモリ内の制限事項を説明します。
