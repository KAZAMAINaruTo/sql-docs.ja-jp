# R のサービスの監視
  使用する場合、このセクションではこれらの一般的な監視タスクに関する情報を提供 [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)]:  
  
-   R で使用されているサーバーのリソースを識別します。  
  
-   R ツールのステータスの監視  
  
-   R のランタイムを使用して実行されるジョブを決定します。  
  
-   使用しているリモートの R のジョブを識別する、 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 計算コンテキストとインスタンス  
  
-   R のジョブと実行時間の長いクエリの関連付け  
  
-   R のプロセスを終了  
  
 インストールされているツールについても説明、 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] R セットアップ、および一覧の Dmv およびに関連する拡張イベントの一部としてコンピューター [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)]します。
