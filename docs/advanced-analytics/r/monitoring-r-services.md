# <a name="monitoring-r-services"></a>R Services の監視
  このセクションでは、[!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)] を使用するときの一般的な次の監視タスクについて説明します。  
  
-   R で使用されるサーバー リソースの識別  
  
-   R ツールの状態の監視  
  
-   R ランタイムを使用して実行されるジョブの決定  
  
-   [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] インスタンスを使用するリモート R ジョブを計算コンテキストとして識別  
  
-   実行時間の長いクエリと R ジョブの関連付け  
  
-   R プロセスの終了  
  
 またこのトピックでは、R セットアップの一部として [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] コンピューターにインストールされたツールについて説明するとともに、[!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)] に関連する DMV と拡張イベントを一覧表示します。
