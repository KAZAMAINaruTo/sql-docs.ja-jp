# ドメイン管理: ドメイン リスト
  ここでのドメイン リストのコントロールについて説明、 **ドメイン管理** ページ [!INCLUDE[ssDQSnoversion](../includes/ssdqsnoversion-md.md)] (DQS)。 このペインを使用して管理操作を実行するドメインを選択します。 すべてのタブ ページで、同じウィンドウが使用される、 **ドメイン管理** ページです。  
  
## オプション  
  
### [ドメイン] リスト  
 **[ドメイン]**  
 この一覧にはナレッジ ベースのすべてのドメインが表示されます。 右側のペインのタブ ページで実行する操作は、この一覧で選択されているドメインに対して実行されます。 詳細については、「  
  
 **複合ドメインの作成**  
 ナレッジ ベースの新しい複合ドメインを作成します。 このコマンドが表示されます、 **複合ドメインの作成** ] ダイアログ ボックス。 このコマンドは、ドメインを右クリックするか、ドメイン リストの上のアイコンをクリックして利用できます。 詳細については、次を参照してください。 [複合ドメインの作成](../data-quality-services/create-a-composite-domain.md)します。  
  
 **ドメインの作成**  
 ナレッジ ベースの新しいドメインを作成します。 このコマンドが表示されます、 **ドメインの作成** ] ダイアログ ボックス。 このコマンドは、ドメインを右クリックするか、ドメイン リストの上のアイコンをクリックして利用できます。 詳細については、「 [Create a Domain](../data-quality-services/create-a-domain.md)」をご参照ください。  
  
 **[選択したドメインのコピーを作成します]**  
 選択したドメインの正確なコピーを作成し、ナレッジ ベースに追加します。 名前は、コピー元のドメインの名前に " - コピー" を追加した名前になります。 このコマンドは、ドメインを右クリックし、使用可能な **コピーを作成**, 、かドメイン リストの上のアイコンをクリックします。 このコマンドは複合ドメインには使用できません。  
  
 **[ドメインをデータ ファイルからインポートします]**  
 .dqs ファイルからドメインをインポートします。 このコマンドを表示、 **データ ファイルからインポート** ] ダイアログ ボックスをファイル システムを参照して、単一ドメインまたは複合ドメインを .dqs ファイルを選択することができます。 このコマンドは、ドメイン リストの上のアイコンをクリックして利用できます。 詳細については、次を参照してください。 [.dqs ファイルからドメインをインポート](../data-quality-services/import-a-domain-from-a-dqs-file.md)します。  
  
 **[ドメインの削除]**  
 選択したドメインをナレッジ ベースから削除します。 このコマンドを表示、 **SQL Server Data Quality Services** ] ダイアログ ボックス。 クリックすると **はい**, 、ドメインとそのすべてのデータの内容が完全に削除します。 このコマンドは、ドメインを右クリックするか、ドメイン リストの上のアイコンをクリックして利用できます。  
  
 **リンク ドメインの作成**  
 選択したドメインにリンクされているドメインを作成します。 このコマンドを表示、 **ドメインの作成** ] ダイアログ ボックス。 このコマンドは、ドメインを右クリックし、をクリックして利用可能な **リンク ドメインの作成** 、選択したドメインにリンクされています。 リンク先のドメインが、[ドメインの作成] ダイアログ ボックスに表示されます。 このコマンドは複合ドメインには使用できません。 2 つのドメインのリンクを解除するコマンドはありません。これを行うには、リンク ドメインを削除します。 リンク ドメインをリンク ドメインに対して作成することはできません。 詳細については、次を参照してください。 [リンク ドメインの作成](../data-quality-services/create-a-linked-domain.md)します。  
  
 リンク ドメインはリンク先のドメインと同じ値を持ちます。 ドメインの名前とプロパティのみが異なります。 リンク先ドメインのドメイン ルール、ドメイン値、参照データ リンク、または用語ベースのリレーションを変更した場合には、リンク ドメインのドメイン ルール、ドメイン値、参照データ リンク、または用語ベースのリレーションも変更されます。 また、リンク ドメインの値を変更した場合には、リンク先ドメインに対しても変更が行われます。  
  
 **[ナレッジ ベースのエクスポート]**  
 ナレッジ ベース全体を .dqs ファイルにエクスポートします。 このコマンドを表示、 **データ ファイルにエクスポート** ] ダイアログ ボックス。 クリックして、このコマンドは使用可能な **ナレッジ ベースのエクスポート データ** または下にある、ページの上部にあるアイコン **エクスポート** ドメイン リスト] ペイン内のドメインのコンテキスト メニューにします。 詳細については、次を参照してください。 [ナレッジ ベースを .dqs ファイルにエクスポート](../data-quality-services/export-a-knowledge-base-to-a-dqs-file.md)します。  
  
 **[ドメインのエクスポート]**  
 ドメインを .dqs ファイルにエクスポートします。 このコマンドを表示、 **データ ファイルにエクスポート** ] ダイアログ ボックス。 このコマンドで使用できる、 **エクスポート** ] メニューまたは [ドメイン リスト] ウィンドウを右クリックして、ページの上部にあるメニュー バーにします。 詳細については、次を参照してください。 [ドメインを .dqs ファイルにエクスポートする](../data-quality-services/export-a-domain-to-a-dqs-file.md)です。
